Показатели: 

   Smartcallback обращение - выводит обращения из сервиса SmartCallBack

   Показы SmartCallBack - выводит показы виджета SmartCallBack

   Посетители (Smartis) - выводит количество посетителей на сайте 


   Группировка: 

   День - группирует данные по дням 

   Месяц - группирует данные по месяцам

   Неделя - группирует данные по неделям

   День недели - группирует данные по дням недели 

   Часы - группирует данные по часам 

   Каналы - группирует данные по рекламным каналам 

   Страница входа - группирует данные по странице входа посетителя

   Устройство - группирует данные по устройствам посетителя 


   Так же, с помощью конструктора показателей, можно создавать новые показатели, используя при этом уже существующие показатели - это могли наблюдать в видео про создание дашбордов.
