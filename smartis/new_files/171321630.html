Краткое видео по разделам интеграции


Для чего необходим

 В данном разделе мы подключаем вообще все интеграции со всеми системами в Smart

Подключенные интеграции - список всех подключенных интеграций по данному клиенту.

Доступные для подключения - здесь менеджер/клиент сможет выбрать доступные для подключения интеграции и настроить интеграции. 

Интеграции с ошибками - перемещаются в верх списка всегда.

В зависимости от сервиса интеграции подключаются по разному. 

Так же в данном разделе можно деактивировать, архивировать или удалить ненужные интеграции.

  • Удаление интеграции - удаление происходит безвозвратно. При удалении интеграции по ней удаляются все данные, в том числе ретро. Новые данные также не будут поступать.
  • Деактивация интеграции - выключение. При этом интеграция не удалится и будет отображаться в списке подключенных интеграций. Деактивация означает, что новые данные по это интеграции больше не будут собираться и синхронизироваться. При этом старые данные останутся в отчетах.

  • Архивация - позволяет скрыть из интерфейса неактивную интеграцию, которая больше не нужна, при этом ретро данные останутся в отчетах. При архивации новые данные по интеграции перестанут поступать. Интеграцию из архива можно вернуть (шаблонная задача в консультации https://app.asana.com/0/1204486395777830/1201657948630287/f). Чтобы архивировать интеграцию, ее нужно сначала деактивировать, потом обновить страницу - появится кнопка "в архив".
     

Имеется еще шестеренка, в  которой по части интеграций можно внести изменения (например, ЯД, ВК) или взять веб-хук для настройки интеграций (Jivo, Marqiz) ,а  по другой части интеграций просто посмотреть информацию по токенам.
