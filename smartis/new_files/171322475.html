Для клиентов, у которых есть своя БД с данными CRM и одновременно нет API для получения этих данных, Smartis может развернуть на сервере клиента приложение, которое позволит выполнять запросы к БД клиента через http(s) запросы.

В каких случаях это нужно клиентам:

  • если есть своя БД с данными CRM и одновременно нет API для получения этих данных.
  • Как пример: Неометрия и Галс

Развёрткой занимается наш, либо разработчик клиента. Документация по сервису и информация о том, как его разворачивать располагается в git.
