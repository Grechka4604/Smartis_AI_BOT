Деление целевых по объектам

В рамках задачи был добавлена математическая функция Делить на количество объектов


Функция позволяет отобразить долю выбранного объекта ко всем объектам, указанным в лиде.
Пример:
Есть CRM Элемент (1267982736), у которого 3 значения в поле Объекты (Режиссёр, Урал, Фреш)
Если выбрать проект Фреш, можно увидеть какую долю он занимает среди всех проектов CRM Элемента.
А именно 0.33%


Если взять Элемент (1267982736), у которого в поле Объекты указан только один (Фреш)
Построив такой же отчет по новому элементу результат будет 1 т.е. Фреш занимает 100% среди всех объектов CRM Элемента


При выборе проекта Все объекты не забываем наложить фильтр
Smartis.CRM: Логика подсчета значения показателя => Количество уникальных значений поля => Объекты
Т.к. нужно подсчитать долю для каждого объекта каждого CRM Элемента.


Кастомный фильтр для показателя Сделок

Задача клиента, которую решает фильтр: Клиент хочет выводить отчет по менеджерам, который будет показывать кол-во лидов, которые бракуются в день обращения. Таким образом клиент и его коллеги будут оценивать эффективность менеджеров 


-В Показателе "кол-во сделок" можно выбрать фильтр "Дата создания vs Доп.поле"


-у фильтра 2 варианта параметра:


  1. "Дата завершения сделки"

  2. "Дата смены ответственного"


При построении показателя кол-во сделок с этими фильтрами, в отчет попадают только те срм элементы, у которых поля соответсвуют условию фильтрации.
