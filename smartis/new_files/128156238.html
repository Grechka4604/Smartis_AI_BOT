Цели:

Повышение лояльности клиента, развитие продукта и донесение качественной пользы за счет постоянных синхронов и прозрачности работ.

Задачи:

В разбор клиента входит:

  • Снятие конверсионных метрик
  • Составление картины по клиенту 
  • Подготовка экспертных предложений на основе общего анализа
  • Ведение карточки клиента

 

Мониторинг и планирование : Все задачи и цикл работы оцифрованы в доске в трелло

Эффективный тайм - менеджмент  - помним, что 2 часа всегда оставляем плавающими на случай коллапсов.

План ведения процесса:

 

Понедельник:

 

Снятие конверсионных показателей :

 

  • Конверсия из посещение в звонок
  • Конверсия из показ звонок
  • Конверсия из посещение в показ
  • Число заявок и показов по типу открытия
  • Анализ мессенджеров -  необработанные, обработанные обращения

 

Вторник/среда:

 

1.Анализ звонков.

Процентное распределение  - Успешные, короткие, заявки на обработку, потерянные.

 

2.Выборочная прослушка звонков на предмет поиска технических ошибок, качества обработки звонка менеджером

 

3. Оценка загрузки КЦ в самые напряженные часы (в том числе со звонками из комеджик/коллтач – через mysmartis)

4. Анализ количества попыток

С какой попытки происходит дозвон, какой процент заявок остается неуспешным по итогу всех попыток

 

5. Анализ работы с чатами/мессенджерами  - распределение по статусам, выборочная проверка качества обработки.

 

Четверг

 

Анализ сайта клиента.

 

1.Проверка всех форм коммуникаций на сайте клиента:

  • Формы обратной связи
  • Заявки на ипотеку/бронирование
  • Email
  • Иконки соц.сетей

 

В каждой форме коммуникации оставить заявку, оценить время обработки и качество обработки.

 

2.Удобство-не удобство популярных разделов. Например, есть ли форма на разделе ипотеки

Проверить бронь, выбор квартиры, как быстро направляют на нужный отдел.

Насколько удобен выбор квартиры, планировок.

 

3. Анализ скорости загрузки сайта

 

 

Также прогон клиента по чек листу возможностей смартиса, на предмет использования всей функциональности сервиса : http://smartis.bi/faq/navighator-po-funktsiiam-omnikanalnoi-sistiemy-obrabotki-obrashchienii-smartcallback/

 

Пятница

 

Формирование отчета с рекомендациями по клиенту на основании данных, полученных за неделю. Валидация с Таней и отправка клиенту.

 

 

После отправки созвониться с клиентом и утвердить список задач в работу.

В 25-26 числах каждого месяца отгружаем клиенту полную картину его текущего состояния по основным метрикам, договариваемся о встрече онлайн для презентации отчета по результатам согласованных задач.
