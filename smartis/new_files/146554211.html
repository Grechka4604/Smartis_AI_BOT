Телефонияпрощупать интерес, предложить нашу телефонию

В настоящий момент у них основной вендор это Манго, которым пользуются корпоративные отделы, у нас они только используют основной номер, который заведен к Манго.

Анатолий не против работы с нами напрямую, запустили процесс по переводу номера в оболочку их кабинета CoMagic.

РМРопределить, есть ли потребностьПотребности нет, ранее велись дополнительные переговоры с Геленой Орловой - руководителем отдела продаж, которая сказала, что собирать детальную статистику по своим менеджерам у нее нет необходимости, есть хватает информации по звонкам внутри CoMagic, эту информацию Анатолий передает в их систему BI, где он уже смотрит. Сам Анатолий подтвердил слова Гелены, что потребности острой нет + это совсем не приоритетный для них проект на данный момент.
FMCпрошупать интерес, предложить наш продуктСо слов Анатолия интерес довольно большой, так как ключевое количество звонков на исходе сотрудники совершают через мобильные устройства. Как итог, с нашей стороны отправлено КП, которое на согласовании у бизнеса.
Сквозная Аналитикаузнать, есть ли потребность, где выстраивают СА сейчасНа данный момент острой потребности нет в продуктах Смартис, тк всю необходимую аналитику они собирают и сводят своими силами в BI системе.
Виджетыузнать, есть ли потребность, можем ли масштабировать использование дополнительного функционала

На анный момент используют стандартные виджеты CoMagic, которые в полной мере закрывают задачу клиента, используют исключительно как инструмент обратного звонка.

За исключением турбо-страницы Яндекса, на который поставить наш виджет, в настоящее время - нет возможности. В итоге мы поставили SCB за 2000 рублей. Который трекает заявки с турбо-страницы и передает их в CoMagic, из которого информация уже попоадает в Amo. Проблемой такого решения является то, что SCB не может передать никакую ифнормацию о метке\источнике обращения в CoMagic, в следствие чего клиент (конкретно маркетолог Александр Ольховский) не видит ценности в таком половинчатом решении.....

Договоренности
  1. По телефонии - ждем подписанный бланк заявления от клиента на перенос номера в ЛК.
  2. По РМР - пока не трогаем.
  3. FMC - ждем решения по КП.
  4. Аналитика - пока не трогаем.
  5. Виджеты - необходимо доработать интеграцию CoMagic c виджетами SCB, т.к. Яндекс заморозил у себя проект по интеграции турбо-страниц с CoMagic. JIRAe3e65afb-eebf-385f-887a-15bc476b2425PRSL-3392
