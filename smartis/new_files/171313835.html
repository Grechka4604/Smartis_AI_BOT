ИНИЦИАЦИЯ ПРОЕКТА

Бизнес-идея (боль) проекта

При проработке проекта "Оптимизация заключения договора с клиентом" было принято решение о смене исполнителя по операционным процессам компании Смартис. Сейчас операционные процессы поддерживаются командой Back-office, планируется, что процессы будут переданы на аутсорсинг или наняты отдельные сотрудники в юридическое лицо Смартис для поддержки данных процессов.

По каким направлениям произойдёт смена исполнителей/передача зон ответственности:

  • юридическая поддержка
  • документооборот
  • биллинг
  • бухгалтерский учёт
  • казначейство
  • кадровый учёт

Подробно артефакты операционной деятельности разобраны здесь

Цели проекта

Что команда должна получить по завершении проекта?

1. До 31.12.2023 спроектированы и согласованы операционные процессы ООО "Смартис" для обеспечения коммерческой деятельности компании в соответствие с её целями/задачами и текущим законодательством.

2. До 31.12.2023 определены ключевые участники процессов (владелец процессов, исполнители, ответственные, участники)

3. До 31.12.2023 согласован план поддержки компании ООО "Смартис" на переходный период.

Задачи проекта

Действие, которое необходимо совершить для достижения вашей цели

Предпроектный этап до 11.10.2023

  1. Описание проекта - цели, задачи, предварительный план
  2. Согласование проекта

Проектирование операционных процессов и согласование плана перехода до 31.12.2023

  1. Изучение текущих процессов
    1. процессы и артефакты
    2. как они работают/требования к выполнению
    3. сколько ресурсов требуется для поддержки
    4. анализ для определения возможность оптимизации (довольны ли мы как оно устроено сейчас)
  2. Проектирование процессов
    a. часть процессов переносим как есть сейчас
    b. часть процессов корректируем с учётом требований/запросов стейкхолдеров
    c. определение ролей участников процессов

3. Подбор исполнителей (выбор, согласование бюджета/договоров)
a. аутсорсинг
b. исполнитель в штат

4. Согласовать план перевода операционных процессов
a. план перехода
b. согласовать договорённости с Back-office на переходный период

Перенос операционных процессов до 28.02.2024

  1. Исполненние плана по переводу операционных процессов

Завершение проекта до 28.02.2024

  1. Фиксация завершения проекта с стейкхолдерами

Продукт проекта

Инструмент для достижения цели проекта

Портфель операционных процессов (работающих) для обеспечения деятельности ООО "Смартис" в соответствие с целями компании и текущим законодательством.

Выгода для компании
  1. Оптимизация ресурса Back-office ООО "НОВОСИСТЕМ".
    a. уменьшаем объём задач, за счёт передачи задач от ООО "Смартис" на аутсорс
    b. не нужно поддерживать два типа процессов - одни для ООО "НОВОСИСТЕМ" и второй тип для ООО "Смартис"
  2. Возможность сделать процессы ориентированные под продукты ООО "Смартис"

Критерии завершения проекта

Какие ключевые точки оценки результатов проекта?

  1. Удовлетворённость результатами проекта всеми вовлечёнными в него сторонами (нематериальный результат)
  2. Операционные процессы ООО "Смартис" выполняются
    a. в соответствие с тем как были спроектированы
    b. исполнители процессов не Back-office ООО "НОВОСИСТЕМ"
  3. Выполнены все требования стейкхолдеров
