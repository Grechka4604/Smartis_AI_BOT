Работа с Whatsapp Buisness API Цель обсуждения

Работа с Whatsapp Buisness API - поняли, что для вас это один из главных приоритетов.
Основные работы по этому треку стартуют в середине февраля, клиентам мы планируем отдать их в конце марта.

Есть несколько нюансов:

1.1. Для заведения и работы с WABA, вероятнее всего, понадобится отдельно подписать договор с Callgear (наше зарубежное юр лицо), в связи с особенностями нашего законодательства. Сейчас этот вопрос решаем с юристами, вернемся чуть позже с уточнением.

1.2. Еще ждем информацию от подрядчика, можно ли будет клиенту принести свой уже зарегистрированный аккаунт WABA для подключения к нам (в ближайшие дни).

Работа с чат-ботами
2.1. Кнопочные / линейные  боты  - данная функциональность доступна уже сейчас, ее мы будем продолжать обогащать, это будет фокус на апрель - май;
2.2. Обучаемые боты  - в рамках этого года мы не планируем самостоятельно развивать такую возможность, но за февраль мы выберем ключевых подрядчиков в этом направлении и в рамках 2 квартала сделаем с ними нативные интеграции.
Дополнительно, у нас подготовлен шлюз, который позволит подключать к нашему сервису любые сторонние системы ботов

Рассылки

4.1. Рассылки в канале Whatsapp будут доступны в конце апреля.

Также, будет доступна возможность каскадной рассылки : (Viber + cмс)  - по ним пока точных сроков назвать не можем, поскольку только занимаемся подвтерждением критичности и важности этого функционала;
4.2. Смс рассылки - модуль смс с рассылками планируем добавить до конца июля;
4.3.Триггерные рассылки с завязкой на сегмент - этот проект только планируется в проработку, со сроками сможем вернуться значительно позже.

Пример каскадной рассылки:
Вы хотите создать каскад, в рамках которого система сначала попытается отправить сообщение клиенту WhatsApp, затем, если сообщение не доставлено в течение 30 секунд, в Viber, и, наконец, через SMS.
