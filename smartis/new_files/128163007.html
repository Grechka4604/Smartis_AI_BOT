Критерии завершения внедрения системы и передачи проекта в поддержку для дальнейшей работы:

  1. Создаем маркетинговые касания по конкретным каналам (ждем список от клиента) для звонков crm по значению из поля "определение рекламы". Проверяем результаты, создаем показатель "все обращения"
  2. Проекты редактируем, чтобы они учитывали страницы объектов на зонтичном сайте.
  3. Протягиваем в договор поле комнатность из объекта недвижимости с условием (для договоров с "типом товара"= "квартира-студия", заменяем значение в поле комнатность с "1" на "Студия")
  4. Выводим фильтр в дашборды (в настройки виджета) для возможности фильтрации показателей (например по типам недвижимости, типам обращений, способе создания интересов) - на оценку 


4 пункт может потребовать больше времени, поэтому, возможно, будет реализован уже в рамках поддержки, но реализован будет, с уточнениями по срокам вернусь после анализа доработки
 
Во вложении закрывающие документы от ноября за первый счет, как и просили, в декабре выставляем и закрываем счет за доработки интеграции с CRM и с января переходим на поддержку.
