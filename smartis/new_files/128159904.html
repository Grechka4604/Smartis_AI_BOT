Условия для клиента

Основная задача: снизить количество звонков с CPC трафика без UTM-меток с 15-25% до 5-10%

Статус

GreenВ работе

Дата согласования плана с Заказчиком

 

Состояние




Предпосылки

  1. ФСК проводят аналитику по UTM меткам, в данный момент они считают точность, и она на уровне 80%, они хотят её довести до 90-95%, так как часть рекламного бюджета не покрывается аналитикой и есть недооцененные РК.


Цели

снизить количество звонков с CPC трафика без UTM-меток с 15-25% до 5-10%. Результат должен быть ежемесячный т.к. нам установили KPI.

Критерии успешности/Требования

звонков с CPC трафика без UTM-меток составляют до 5-10%


Есть 2 KPI: 

Учитываем только рекламные кампании, где есть UTM разметка - понимаем по настройкам РК. 

Учитываем только рекламные кампании с номерами ДТ >0

  1. % звонков с utm на статику+динамику от всех звонков на динамику - 95%
  2. % звонков с utm на статику+ динамику от всех звонков - 90% 


Звонок без сессии, но он звонил ранее, то мы должны подтягивать utm. - Сейчас это не так. 

Звонок без сессии на номер ДТ, то окей, можем не подтягивать utm.


Методология от агентства Ай Проспект (Эдгар Дихтярев):https://prnt.sc/pwbpft

Этот отчет строится на основе всех данных из комеджика, по всем сайтам и кампаниям.

Кампании, у которых только статичные номера также входят в этот отчет, но выбрав сверху в фильтрах площадки яндекса и гугла, мы получим более репрезентативные данные, т.к. по этим площадкам используются кампании с динамическими номерами.

Состав работ

  1. Подготовить отчёт, который позволит показать текущую ситуацию по выполнению KPI.  CCV-253
  2. Сделать доработку через ОПР, которая позволит подтягивать UTM метки с предыдущих сессий PRSL-1523
  3. Повторно сформировать отчёт, чтобы понять каких результатов достигли после доработки в пункте 2. https://jira.uiscom.ru/browse/CMPM-1213
  4. KPI достигнут? 
    1. ДА - передаём информацию клиенту, получаем от клиента ок; 
    2. НЕТ - дополняем план работ, что ещё необходимо сделать, чтобы KPI был выполнен?
