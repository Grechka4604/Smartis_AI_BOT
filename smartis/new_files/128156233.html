Цели:

Повышение лояльности клиента, развитие продукта и донесение качественной пользы за счет постоянных синхронов и прозрачности работ.

Задачи:

В разбор клиента входит:

 

  • Анализ клиента по основным метрикам «здоровья аналитики», а также мониторинг качества работы бизнес-процесса клиента.
  • Составление плана развития по найденным в результате анализа точкам роста
  • Согласование проекта/списка работ с клиентом
  • Встреча с клиентом и презентация результатов работы

Мониторинг и планирование :

Для анализа клиента используется внутренний дашборд, включающий в себя следующий набор параметров:

•               Матчинг по основным метрикам воронки, а также матчинг обращений

•               Динамика трафика, обращений, конверсиий (причем обращений общих и целевых)

•               Динамика визитов, сделок, конверсии из обращений в визиты и сделки

•               % клиентов у которых касаний в карточке после обращений больше n-числа

•               Анализ ботов в сессиях по источникам  - как следствие работа с этими аудиториями

•               Анализ расходов  - расходы по площадкам  (обращения как маячок адекватности присутствия/отсутствия расходов)

•               Срез сделок  по объектам

•               % несостоявшихся сделок, а также причины отказа  - если нет - то обязать заполнять, чтобы активно работать с аудиторией

•               Средняя стоимость по объектам/ каналам

•               Цикл сделки  - настраиваемый и с привязкой к окну ретроспективы

•               Визиты  - соотношение состоявшихся  / не состоявшимся, их их срез по объектам

•               Анализ LTV и повторных продаж

 

 

?Доп. Проект: Провести аудит средних стоимостей по классу жилья и площадкам/каналам - добавить в кабинеты клиентов как системные показатели скрытые + цикл сделки средний по больнице по аналогии

Починить перетекания, а именно  - настройку этапов и метрик (она не сохраняется сейчас) ?

 

Дополнительно проводится работа по анализу сделок, на основании Дашборда, включающего:

  • Сделки по типам - квартиры/парковки/кладовки и соотношение сумм сделок по ним

А это значит в качестве основы для этого анализа просить клиента давать инфу по внешним факторам : темп строительства, мин цены и т.п

 

  • Анализ сделок по планировкам, комнатности

С каких каналов лучше продается та или иная комнатность

 

  • Анализ по менеджерам  - % открытых, проваленных, закрытых  Соотношение кол-ва и сумм сделок

Эффективный тайм - менеджмент  - помним, что 2 часа всегда оставляем плавающими на случай коллапсов.

 

После подготовки утвердить список задач с клиентом в работу и зафиксировать в клиентской доске в Trello.

В 25-26 числах каждого месяца отгружаем клиенту полную картину его текущего состояния по основным метрикам, договариваемся о встрече онлайн для презентации отчета по результатам согласованных задач.
