Клиент может обратиться с требованием/письмом о необходимости предоставить документы по ООО «СМАРТИС», к примеру, в целях проверки благонадежности нашей компании или проверки полномочий лиц, подписывающих от имени компании документы и т.п.

Предоставление клиенту документов (в виде скан-образов) осуществляется через постановку задачи в Jira.

1. Постановка задачи (зона ответственности менеджера)

  • менеджер формирует задачу в Jira на специалиста по документообороту (Smartis Документооборот (SMA))

Тип задачи: задача

Тема: предоставление документов клиенту <наименование юр. лица>

Юридическое наименование клиента (кратко): <наименование юр. лица>

Тип документа: Иной запрос (счет, закрывающие, учредительные, письма и т.д.)

Описание: клиент запросил документы по нашей компании, перечисляет документы, которые запросил клиент ЛИБО вкладывает в задачу требование/письмо клиента

  • специалист по документообороту берет задачу в работу (переводит задачу в статус «Подготовка документа») и тэгает ведущего юриста с комментарием «Проверить возможность предоставления документов клиенту по требованию»


2. Согласование предоставления документов по требованию клиента (зона ответственности ведущего юриста)

  • ведущий юрист проверяет возможность/законность предоставления документов клиенту

1) если все запрашиваемые клиентом документы могут предоставлены — тэгает специалиста по документообороту с комментарием «Все документы могут быть предоставлены»

2) если часть документов не может быть предоставлена клиенту (документы содержат персональные данные какого либо субъекта персональных данных либо содержат конфиденциальную информацию и т. п.) - тэгает специалиста по документообороту с комментарием «Часть документов не может быть предоставлена», перечисляет каждый документ, который не может быть предоставлен с указанием причины отказа в его предоставлении


3. Формирование пакета документов (зона ответственности специалиста по документообороту)

Специалист по документообороту

  • вкладывает в задачу документы в объеме, указанном ведущим юристом в п.2 настоящего регламента, тэгает менеджера с комментарием «Документы подготовлены, которые могут быть предоставлены»

  • в отношении документов, которые не предоставлены делает менеджеру ссылку на комментарий ведущего юриста

  • закрывает задачу
