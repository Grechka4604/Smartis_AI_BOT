Способы визуализации отчетов

  1. Нажать крайнюю левую иконку в видах отчетов: комплексный отчет
  2. Вторая слева иконка: график
  1. В комплексном отчете отображаются график, таблица и может быть еще диаграмма (если при выбранных параметрах отчета есть что разделять на сегменты диаграммы). 
  2. При выбранном типе визуализации "График" отображается только график (нет диаграмм, нет таблиц)
2
  1. Выбрать "кольцевая диаграмма" (третья иконка слева)
  2. Выбрать временнУю группировку (день, неделя, месяц)
  3. Выбрать только один показатель
Вид визуализации не применяется (будет пустое поле отчета)
3
  1. Выбрать "кольцевая диаграмма"
  2. В отчете выбрать один показатель
  3. Выбрать группировку (канал, площадка, кампания, город или регион)
Диаграмма отобразит доли признака, который указан в группировке, для этого показателя
4
  1. Выбрать "кольцевая диаграмма"
  2. Выбрать группировку по времени
  3. В отчете выбрать более одного показателя (из тех, что можно сравнить - например: сделки+бронирования)
Диаграмма отобразит соотношение по долям между выбранными показателями
5
  1. Выбрать группировку гео (регион)
  2. Выбрать показатели: CRM, обращения или посетители сайтов
  3. Кликнуть иконку карты
  4. Навести курсор на карту
Построится визуализация карты. При наведении курсора на карту, будет отображаться значение показателя в выбранном регионе. Регионы, где больше всего данных, на карте темнее.
6
  1. Выбрать визуализацию "таблица", построить отчет
  1. отчет будет визуализироваться только в виде таблицы
7
  1. Выбрать "одноцифровой" вид визуализации
  2. Выбрать несколько показателей в отчете
Для каждого выбранного показателя отображается отдельный одноцифровой виджет
8
  1. Построить отчет по данным, по которым недоступна детализация
  2. Выбрать вид визуализации "исходные данные" (самая правая иконка)
При попытке использовать такую визуализацию для показателя, по которому нет исходных данных, отобразится ошибка: При построении отчета возникла ошибка. Для таблицы smartcallback_ch2.insights_ch1 отсутствует класс реализации получения исходных данных
9
  1. Выбрать в отчете несколько показателей с одним типом данных (например, CRM или сервисы обращений)
  2. Выбрать в отчете несколько показателей с разными типами данных (показатели и CRM, и из сервисов обращений и другие)
  3. Выбрать в отчете показатели с разными типами данных + показатели, по которым исходных данных не существует
  1. В Исходных данных будут все данные из выбранных показателей на одной странице
  2. Под данные каждого типа создастся отдельная вкладка в отчете (например, вкладка для данных по обращениям)
  3. В таком случае отчет не построится и появится ошибка: При построении отчета возникла ошибка. Для таблицы smartcallback_ch2.insights_ch1 отсутствует класс реализации получения исходных данных
10
  1. Выбрать визуализацию "таблица", построить отчет
  2. Выбрать остальные виды отчета с теми же условиями
Проверить, что все виды отчетов отображают одно и то же итого, как в табличном: проверить на CRM-показателях, на групповых, по обращениям и по сессиям
11

Дополнительные действия с отчетом

  1. Построить отчет, выбрав отображение "Таблица" или "Комплексный отчет"
  2. Кликнуть на иконку "палитра"
  3. Выбрать столбец в отчете
  4. В списке выбрать способ:
    a. Градиент от min к max
    b. Градиент от min к max (негативный)
    c. Скрыть столбец
    d. Скрыть строку
    e. Условное форматирование

2. при клике на иконку добавляется одно условие форматирования

3. откроется выпадающий список с видами форматирования

4. a. добавит к значениям в отчете градиент, максимальные значения будут выделены ярче 

b. добавит к значениям в отчете градиент, максимальные значения будут выделены ярче

c. выбранный столбец будет скрыт из отчета

d. позволяет скрыть из отчета из выбранного столбца строку или строки , значения которой соответствуют заданному условию.

e. позволяет раскрасить столбец и его значения по заданным условиям. 

12
  1. Создать отчет с возможными вариантами форматирования
  2. Сохранить отчет
  3. Открыть сохраненный отчет
Все форматирования сохранятся в отчете
13
  1. Построить отчет "Таблица" или "Комплексный отчет"
  2. Иконка "Стрелка" - скачать отчет
  3. Построить отчет "Исходные данные"
  4. Иконка "Экспорт"
  5. Выбрать формат - скачать отчет
По клику на иконку, отчет скачается для просмотра файла в формате таблицы
14
  1. Построить отчет "Таблица" или "Комплексный отчет"
  2. Иконка "Распечатать"
Отправит отчет на печать
15
  1. Построить отчет "Таблица" или "Комплексный отчет"
  2. Иконка "Транспортировать"
В отчете столбцы и строки поменяются местами: показатели, которые были в строках, будут отображаться в виде столбцов и наоборот.
16
  1. Построить отчет "Таблица" или "Комплексный отчет"
  2. Иконка "Отклонение от среднего"
  3. Еще раз нажать на иконку

2. Появится визуализация средних значений

3. При повторном нажатии визуализация отключится

17

Настройка отображения данных в таблице

  1. Построить отчет → Нажать "показывать по..."
  2. Выбрать диапазон
В отчете показывается выбранное количество строк
18
  1. Построить отчет → В строке поиска по таблице ввести значения поиска (буквы латинницы\кириллицы, числовые значения, символы)
  2. Проверить итоговое значение

Найдутся и покажутся все строки, содержащие введенные значения, без обновления страницы

Итоговое значение при сортировке строк меняться не будет

19
  1. Ввести в строку поиска невалидные значения, либо отсутствующие в отчете символы
Записи отсутствуют
20
  1. Построить отчет → Кликнуть на иконку возле названия показателя (стрелки вверх и вниз) - сортировка по возрастанию
  2. Повторно кликнуть на иконку - сортировка по убыванию
  1. когда стрелка указывает вверх, таблица сортируется по выбранному столбцу, при этом наибольшие значения выбранного столбца располагаются вверху
  2.  когда стрелка указывает вниз, таблица сортируется по выбранному столбцу, при этом наименьшие значения выбранного столбца располагаются вверху
21
  1. Построить отчет → необходимо навести курсор на строку в таблице
  2. Кликнуть на иконку
  3. Выбрать нужную детализацию
  4. Снова кликнуть на иконку стрелки и выбрать детализацию
  5. Нажать на иконку напротив строки (стрелка вверх)
  1. В строке появится иконка стрелки вниз
  2. Появится выпадающий список с детализациями
  3. В отчете отобразится информация по выбранной детализации
  4. "Провалимся" в следующий уровень детализации
  5. Детализация будет скрыта
22
  1. Построить отчет выбрав показатель "Все обращения"
  2. Выбрать группировку "день"
  3. Выбрать период - 2 месяца
  4. Построить отчет и пролистать вниз результат
При пролистывании строка заголовков и строка итогов - закрепляется
23
  1. Построить отчет в виде таблицы

  2.  Перестроить столбец в одноцифровой виджет

При перестроении отчета отображается корректное значение, применяется округление
