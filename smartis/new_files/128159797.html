Цели:

  • активный выход на регионы
  • подключить компании из интерпрайз, которые нам интересны и пока не с нами
  • отраслевое решение продвигать на международный рынок

Что можем предложить с точки зрения партнерства:

  • понимаем глубоко проблематику клиентов рынка/коммуницируем на разных уровнях/сильный маркетинг
  • сотрудничество на уровне продукта
  • кросс продажи (рекомендации)

продуктовая линейка ТИСА

понять, какие направления есть у ТИСА и что развивают

Работают на базе MДайнемикс

Сейчас собирают облачное решение совместно с майкрософт: облако в России. Развернули, когда была проблема с облаком за рубежом. Создано для выхода за пределы Мск и МО.

Проблема с выходом на регионы ТИСА: стоимость важнее качества/не доросли. Поэтому принято решение выходить с чем-то готовым. На этом стыке и рождается наше партнерство.

!!! Сейчас используется живо сайт при интеграциях. Недостаток: клиент контактирует с 1-ым эшелоном, (типа КЦ), а далее — коммуникация уже теплого лида недоступна (2-ой эшелон). У ТИСА есть своя разработка с бизнес-вотс ап, почта, а также с тимс для взаимодействия менеджера и передачи данных в реальном времени.

Схема партнеров сегодня:

1-ый эшелон

  • Простые звонки: для интеграции с множеством станций
  • КТ- коллтач/км/?(кто-то)
  • ЖИВО сайт коммуникация для конакт центрови

Среда хранения срм

2-ой эшелон

  • бизнес-вотсап внутри СРМ
  • интеграция с тимс — для быстрого обмена информацией между сотрудниками

Решения:

  • своя агентсткая сеть. Позволяет коммуницировать с брокерами. Иными словами — это фиксация клиента у застройщика: задача брокера быстрее зафиксироваться у застройщика на покупку квартиру, чтобы получить маржу.
  • Работа с пропущенными: они отображаются отдельным списком для обработки
  • Аналитика через Пауэр биай
  • Система расчета сделок, формирование вордовских док-ов (генерация ДДУ автоматически).
  • Море готовых интеграций, в т.ч. сбер и др.эквайринги для онлайн продаж
  • видео-звонок прямо с сайта (уже продвигают через майкрософт)
  • ФИЧА! окно, в котором менеджер отчитывается за предыдущий шаг и ставит новый с привязкой часового пояса клиента! такого в России ни у кого нет.

Конкурировать с АМО и профитбейс не хотят, но будут по стоимости дешевле., за счет перезода на офис 365 с майкрософта, так как облако закрывает большее кол-во потребностей. Создали веб срм на базе пауэрапс, т.к. собственная разработка + веб версия в 7 раз дешевле майкрософт.


По этому проекту Майрософт будет помогать продавать.+лейбл, маркетинг.
