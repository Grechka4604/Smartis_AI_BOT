Раздел, благодаря которому можно создать Smartis пиксель Post-view в ручную из интерфейса, раньше это можно было сделать только через разработку. 

Интерфейс

  1. Поиск по конкретному пикселю.
  2. Окно создания пикселя.
  3. Площадка, к которой относятся пиксели.
  4. Название кампании.
  5. Название объявления.
  6. Формат, картинка объявления.
  7. Кол-во просмотров по данному объявлению.
  8. Экспорт просмотров по пикселям. 
  9. CTR.
  10. Адрес посадочной странице.
  11. Кто из пользователей создал.
  12. Пиксель на показ.
  13. Пиксель на клик.


Создание пикселя

Перед созданием пикселя проверьте работоспособность ссылки, которую вам дали для создания пикселя:

- в ней не должно быть пробелов
- При переходе по ней, сайт должен нормально открываться и вести на нормальные, существующие страницы (не 404)


  1. Нажав на кнопку "Создать пиксель", откроется модальное окно создания пикселя. Все поля являются обязательными для заполнения, на что указывает знак '*'. 

  2. Заполнить данные. 

               

3. Сопоставить занчения из google doc. с модальным окном, где Заголовок Объявления - это Наименование Банера, Наименование формата размещения - Наименование формата медийки и Полный адрес посадочный страницы - Ссылка, куда должен вести файл с метками.  

4. На картинку действуют ограничения по форматам, можно загружать только jpg,jpeg,png,gif и по размеру не более 10Мб 

5. Создаете пиксель и копируете, добавляете в google doc. 


Или сделать выгрузку: 

В экспортируемый файл попадают все поля из таблицы, небольшими отличиями, в значениях полей "Пиксель на показ" и "Пиксель на клик" будут ссылки вместо подписей "Пиксель на показ" и "Пиксель на клик", а объявление будет состоять только из заголовка объявления, без картинки и описания.


Замена ранее созданного пикселя


В ситуациях, когда клиент обновляет ссылки для ранее созданных пикселей, нужно просто заменить ссылки в настройках, а пиксель не менять и не делать новый.


Чек-лист поддержки smartis пикселей с клиентом и агентством. 

  1. Все метки заносятся в файл Google таблицу(у каждого клиента и агентства своя). *Обязательно проверить ссылку на картинку, чтобы она открывалась. 

  2. Далее агентство пишет Smartis запрос на название площадки.

  3. Smartis выделяет существующую площадку или создает новую в коробке/облаке.  

  4. Агентство создает пиксель через личный кабинет Smartis (облако). 

  5. Агентство отдает пиксель на площадку, контролирует корректность установки пикселя. 

  6. Через 2-3 дня агентство сверяет данные по просмотрам в облаке. 

  7. Далее через неделю сверяем корректность по кол-ву просмотров.
