Цель - Хранение информации по клиенту в едином пространстве для более быстрого и удобного погружения новых менеджеров в проект. 

Участники процесса: Project менеджер внедрения.

Инициатор по Asana: Project менеджер внедрения.

Инфо поле: Asana.

Необходимо фиксировать всю основную информацию по клиенту в основном проекте asana или сначала в проекте внедрения, потом перенести при передаче в основной проект.


Основной проект в Asana создается по шаблону https://app.asana.com/0/1206889317970446/1206897054147969

Название пишем "Бренд / Основной проект" и добавляем в портфель клиента. 


В основной проект в asana фиксируем:

  • Контактные лица (ФИО) (Должность Номер телефона Почта Телеграмм)

    ЛПР:

    Маркетолог:

    Разработчик со стороны клиента:

    NDA

    - да/нет

    Описание компании

    - наименование

    - регион

    - отрасль (авто/недвижка/брокер)

    Документооборот\специфика:

    - инф. письмо (да/нет)

    - способ отправки (курьер, ЭДО, почта РФ, заказное письмо)

    - оплата (по сканам/оригиналам)

    - отсрочка платежа (да/нет (количество дней)

    - особые условия сотрудничества (если по клиенту есть индивидуальные договорённости по работе, прописываем в данном пункте)

    РА (если есть список РА и КЛ)

    - .......

    Коробка(лицензия)/облако

    - коробка/облако

    Храним в Asana:

    Цель проекта внедрения Smartis ?.......

    Какой набор отчётов используют ?......

    В каком процессе используют ? ......

    * В вложение в проект по клиенту добавляем:

    - технические задания по доработкам (версия согласованная с клиентом)

    - заполненный бриф

    - прочие файлы/артефакты 

    Шаблон описания https://app.asana.com/0/1206864152384341/1206875756181594/f


Важно! Заполненную карточку в Conf мы принимает от отдела продаж, без нее проект не принимается (описано в регламенте передачи)!


В карточку confluence фиксируется:

  • Менеджер внедрения
  • Менеджер поддержки
  • Bizdev
  • ID Infopin
  • ID Партнёра (например, ЕРЗ ID 875018 )
  • Ссылка на проект в Asana
  • Пользователей
  • Количество ЖК
  • Регион
  • Бюджет на маркетинг и рекламу (хотя бы digital)
  • CRM
  • Где фиксируются сделки и сумма сделок
  • Сервисы колл-трекинга
  • Виджеты
  • Телефония
  • Цель проекта
  • Какую боль лечим?
  • Продукты
  • Дашборды и отчеты в кабинете
  • Согласованное КП
  • Ключевые особенности работы с клиентом
  • Чек лист по документам
  • Контактные лица
  • Брифы, ТЗ
  • Основное - отчет по офлайн встречам!
