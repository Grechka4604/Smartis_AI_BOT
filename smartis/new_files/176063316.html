Дашборд по Post-view представляет собой виджеты, которые могут помочь с:

  1. Отслеживанием эффективности медийных площадок до кампаний/объявлений;

  2. Отслеживание конверсий на всех этапах воронки с учетом Post-view и без учета Post-view ;

  3. Автоматизацией  подсчета целевых метрик с учетом просмотров медийных размещений - целевых обращений, визитов, броней, сделок. 


  1. Отчет по Post-view представляет из себя дашборд, который состоит из нескольких виджетов. 

Желательно рассматривать итоги по Post-view на периоде трех месяцев, не включая месяц построения, это обусловлено тем, что цикл от просмотра до сделки гораздо больше, чем от целевого действия до сделки. 

Виджет по динамике данный по просмотрам за выбранный период, нужен для отслеживания поступления данных по просмотрам 

Виджет содержит в себе показатель: 

  • Просмотры медийных размещений 


2. Виджет для медиапланирования. Используется для оценки эффективности каждой площадки, на которых размещается медийная реклама.

Это основной отчет, который помогает определить эффективность использование данных площадок на периоде, выбранном в календаре.

Виджет содержит в себе показатели: 

  • Бюджет факт с НДС и АК

  • Просмотры медийных размещений

  • CoMagic звонки

  • CoMagic звонки (по моделям аттр.)

  • CoMagic заявки

  • CoMagic заявки ( по моделям аттр.)

  • CRM: Целевые обращения без post-view

  • CRM: Целевые обращения с post-view

  • CPA с post-view

  • CRM: Сделки без post-view

  • CRM: Сделки с post-view

  • ДРР с с post-view

  • ROMI


! Важные заметки по показателям, которые участвуют в отчете:

  1. Если у вас обращения поступают только через конкретный коллтрекинг, в отчете достаточно выбрать только показатели с него

Если обращения у вас поступают из коллтрекинга и сервизов квизов/заявок/чатов, то в отчете нужен будет показатель : 

  • Все обращения без post-view

  • Все обращения с post-view

  1. Мы используем показатели по заявкам: 

  • CoMagic заявки

  • CoMagic заявки ( по моделям аттр.)

как подтверждение реальности трафика с некоторых площадок, так как многие рекламные площадки накручивают звонки, но мало кто накручивает заявки 

  1. Воронка клиентов может быть шире, в данном отчете показаны этапы воронки, которые использует клиент, поэтому основными показателями являются: 

  • CRM: Целевые обращения без post-view

  • CRM: Целевые обращения с post-view

  • CPA с post-view

  • CRM: Сделки без post-view

  • CRM: Сделки с post-view


Работа с данным отчетом

  1. Смотрим на показатели : 

  • CoMagic звонки

  • CoMagic звонки (по моделям аттр.)

Выбираем топ площадок, которые имеют явный эффект от использования медийной рекламы, фиксируем их для себя.

  1. Смотрим на показатели 

  • CoMagic заявки

  • CoMagic заявки ( по моделям аттр.)

Стараемся также выделить топ площадок, которые имеют явный эффект от использования медийной рекламы, фиксируем их для себя. Параллельно сравнивая со списком из пункта 1.

  1. Смотрим эффект на первом этапе воронки, используя показатели: 

  • CRM: Целевые обращения без post-view

  • CRM: Целевые обращения с post-view

Используем ту же методику, как и в предыдущих этапах. Выбираем топ площадок, которые имеют явный эффект от использования медийной рекламы, фиксируем их для себя. Параллельно сравнивая со списком из пункта 1 и 2. 

  1. Смотрим эффект на втором этапе воронки, используя показатели: 

  • CRM: Сделки без post-view

  • CRM: Сделки с post-view

Используем ту же методику, как и в предыдущих этапах. Выбираем топ площадок, которые имеют явный эффект от использования медийной рекламы, фиксируем их для себя. Параллельно сравнивая со списком из пункта 1,2,3. 

После формирования топа площадок, которые на всех стадиях коммуникации с клиентом показали эффект, можно переходить:
1. К оценке количества просмотров медийных размещений, по топовым площадкам

2. К оценке бюджета, потраченного на данные размещения медийных баннеров 

3. К оценке конечно доли рекламных расходов (если необходимо, то еще и оценке ROMI)

Далее каждую топовую площадку можно рассмотреть отдельно в динамике. Для этого нужно : 

  1. Нажать на необходимую площадку 

  2. Подняться к Дашборду по динамике
