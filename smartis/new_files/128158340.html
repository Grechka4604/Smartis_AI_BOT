1.2. Чек- листы.

Для контроля качества сотрудников мы предлагаем использовать отчет Чек-листы. Внутри личного кабинете клиент сможет видеть процентное соотношение звонков с соблюдение всех блоков разговора в разрезе по сотрудникам. В связи с тем, что переадресация идет на один сип транк, такую настройку система распознает как одного сотрудника, мы ждем релиз на нашей стороне. Данная доработка позволит подгружать информацию о сотрудниках из системы клиента по API и соотносить их с разговорами.
