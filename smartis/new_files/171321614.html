Что такое?

База обращений - это особый раздел для просмотра исходных данных по сервисам обращений (другие исходные данные он не показывает)
База обращений находится по ссылке: https://my.smartis.bi/base-requests, также ее можно открыть из бокового меню Smartis (раздел "база обращений")

Данный раздел помимо просмотра исходных данных по сервисам обращений за выбранный период также имеет ряд дополнительных функций:

Какие сервисы отображаются? 

  • Все сервисы коллтрекинга. ссылка на сервисы    

  • Сервисы квизов/чатов/ заявок. ссылка на сервисы
  • Маркетинговые касания CRM. Ссылка 

Интерфейс

Добавление фильтра: работает также, как фильтры в Конструкторе отчетов (можно почитать тут: ). По клику на кнопку "добавить фильтр" появляется список доступных фильтров. Можно накладывать несколько фильтров одновременно. Добавленные фильтры отображаются в правой верхней части экрана.




Выбор проекта: работает также, как в Конструкторе отчетов (можно почитать тут: ). Можно выбрать сайт, объект или проект, по которому нужно посмотреть исходные данные по обращениям.

Если вы добавили фильтры и/или изменили проект, чтобы обновить отображаемые данные, нужно нажать "Применить фильтры"



Экспорт данных: доступен по клику на кнопку Экспорт, можно скачать данные в формате CSV или XLSX.


Поделиться отчетом: позволяет отправить ссылку на построение в Базе обращений (если нужно поделиться с пользователем Smartis) или отправить снимок отчета (если нужно отправить отчет не-пользователю Smartis)



Также в Базе обращений можно Добавить столбцы к отображаемым данным. Это описано тут:  в разделе Информация в исходных данных

Какие исходные данные доступны в базе обращений - читать тут:  в разделе Информация в исходных данных
