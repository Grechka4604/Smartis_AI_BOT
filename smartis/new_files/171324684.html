Дни в отчете кликабельны - если кликнуть на какой-либо день, то отобразится информация о сервисе справа на странице:

  • кол-во интеграций с сервисом
  • статусы по подсчету достоверности


  1. Кликнуть на любой из сервисов

Название сервиса кликабельно - по клику на него открывается меню справа. Оно содержит информацию об активных интеграциях с сервисом, синхронах по этому сервису и окно обновления данных

  1. Кликнуть на иконку стрелки слева от названия сервиса
Откроется список интеграций
  1. Кликнуть на один из дней в открытых интеграциях

По клику на день открывается справка справа на странице:
Здесь указан: 

  • общий % достоверности по этой интеграции
  • подробная информация о достоверности, в разбивке по сущностям, которые получаем из интеграции. Например, для АМОCRM - по сделкам и контактам. Для разный сервисов набор будет отличаться
  • когда обновляли достоверность по этой сущности за выбранный день
  • сколько данных у нас
  • сколько данных в API (во внешней системе)
  1. Открыть список интеграций
  2. Проверить, есть ли среди них интеграции с ошибкой
По таким интеграциям достоверность не отображается с момента, когда мы получили ошибку, они выделены красным шрифтом
  1. Открыть детализацию по интеграции
  2. Просмотреть информацию о синхронах

В информации по синхронам доступны такие данные:

  • id синхрона
  • название синхрона
  • периодичность его работы
  • статус (активен/не активен)

  1. Обновить достоверность + данные 
Данные добавятся в очередь на обновление и отобразятся в выпадающем списке.

Когда обновление завершено, статус будет "Данные: завершено, Достоверность: завершено". Это значит, что данные обновлены и достоверность за нужный период пересчитана.
