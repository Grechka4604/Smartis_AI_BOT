SmartCallBack может произвести интеграцию с контекстной рекламой:
нужно будет подключить интеграцию с рекламными кабинетами в личном кабинете Smartis, с помощью которой в пару кликов подтянем все контекстные объявления с площадок (YandexDirect, GoogleAds).
Когда посетитель будет переходить по этим объявлениям (контекстной рекламы) - мы будем показывать определённый текст в виджете, который связан по смысловой нагрузке с контекстным объявлением.
Для начала, надо подключить интеграцию с рекламным кабинетом, GoogleAds и(или) Яндекс.Директ в кабинете Smartis. 
Нажимаем "Администрирование" → "Интеграции"




В разделе с интеграциями ищем таблицу "Доступные для подключения", в ней находим GoogleAds / Яндекс.Директ, нажимаем подключить.
Подключить интеграцию может тот человек, у которого есть доступ в подключаемый аккаунт!
Поэтому, подключение производит либо сам клиент/РА, либо нам выдают логин:пароль и мы сами производим подключение.




После того, как подключили интеграцию - её нужно привязать к соответствующему домену. 
Заходим в "Администрирование" → "Проекты и сайт" → вкладка "сайты" → ищем в списке нужный домен → жмякаем на него 






В настройках домена есть блок: "Интеграции доступные для привязки", в котором отображаются все подключенные интеграции, которые находятся в разделе "Администрирование" → "Интеграции".
Выбираем нужную интеграцию, подключаем её.







Далее нужно обновить данные, подтянуть их из рекламных кабинетов.
Заходим в раздел "Конструктор отчетов":
-выбираем проект
-выбираем показатель Google Ads клики или Я.Директ клики, в зависимости от того, что нужно обновить
-ставим группировку "день"
-ставим вчерашний день
нажимаем построить.
Загрузились данные, далее появились кнопки "обновить статистику Яндекс Директ" и "обновить статистику Adwords".
Нажимаем их по очереди, но не одновременно! Сначала обновили статистику одного рекламного кабинета, затем другого.
Пока крутятся стрелочки - обновление не закончилось. 



После того, как всё обновилось - идём в настройки домена:
"Администрирование" → "Проекты и сайты" → вкладка "сайты" → находим нужный сайт, жмякаем на него, переходим во вкладку "SmartCallBack" → далее переходим во вкладку "тексты" → открываем раздел "контекст" 
Если у вас объявления загрузились только с заголовком, а основной посыл заложен в подзаголовке - то нужно применить настройку "группировать по текстам объявлений".
В настройках 


Будьте внимательны! На одном рекламном аккаунте могут быть рекламные объявления сразу по нескольким ЖК/проектам. 


Выбираете нужное объявление, пишите заголовок + подзаголовок (HTML теги можно применять), сохраняете, появляется кнопка "посмотреть"



С помощью этой кнопки вы сможете сымитировать переход по рекламному объявлению и проверить, отображается ли нужный текст.
Текст покажется только один раз, при повторных открытиях будет отображаться текст и версий настроек.
Не забывайте включить ползунок, чтобы текст начал работать текст.
