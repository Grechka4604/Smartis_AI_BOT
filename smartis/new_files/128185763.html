  • Есть общий системный конвеер поствью данных в который нужно встроить DCM методы
  • Есть общее решение вставки поствью-пикселей в сайты наших клиентов (floodlight-теги), но самой генерации тегов и вставки в сайты нет

Для добавления данных по просмотрам в Smartis, возможно использовать ручную выгрузку данных из DCM (разметка и выгрузка из DCM производится на стороне клиента). 

Файл для импорта данных в Smartis должен соответствовать данному шаблону. 

Перед импортом в Samrtis, его необходимо конвертировать в csv формат. После конвертации файл можно загрузить в Smartis, сопоставив поля согласно скриншоту ниже:
