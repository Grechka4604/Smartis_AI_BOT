Вопросы по PPVVC и BANT:

Основные вопросы по PPVVC тут

  • Какие задачи в ближайшей перспективе/долгосрочные хотите решать?
  • На какие метрики опираетесь для решения задач?
  • Есть ли конечная цель в цифрах? (Например, снизить CPA на 3% до конца квартала)
  • Кто принимает решение о приобретении СА?
  • Кто конечный пользователь системы? (кто может помочь с тех стороной вопроса?)
  • Есть ли департаменты/отделы, которые будут участвовать в согласовании/работе? (техспецы, IT спецы, маркетологи и тд)
  • Решает ли такой подход/отчёт/дашборд вашу задачу?
  • Value подтверждается в процессе сверки по блокам УТП
  • К какому сроку задачи должны быть решены? (должна быть внедрена аналитика)
  • Какой бюджет на digital в мес?
