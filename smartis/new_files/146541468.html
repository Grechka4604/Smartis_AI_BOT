Постмит встречи:

  1. Детализация по звонкам за 2021 год следующая
    Всего за 2021 год было зафиксировано 465 386 звонков. В это число входят абсолютно все типы звонков (Потерянные, Принятые и СПАМ): Входящие, Исходящие, Call API и т.д.

    Доля СПАМа от общего вала составляет 39 883 звонка. Т.е. - 8,6% (такие звонки мы отсекаем до начала момента Вызова. Т.е., не даем им попасть в сеть, а просто фиксируем факт попытки сделать Вызов).

    Всего (без учета СПАМа) получено 425 503 обращения.

    Кол-во Входящих звонков (без СПАМа) - 189 368.

    Из них:

    • принятые (операторы подняли трубку) - 130 288 звонков.

    • потерянные (операторы не подняли трубку, либо абонент сбросил звонок на стадии Инф. сообщения) - 59 170 звонков.

    Кол-во Исходящих звонков (включая автоматические перезвоны по Call API) - 236 045 звонков.

    • успешные (состоялся контакт с абонентом) - 127 110 звонков.

    • абонент не ответил - 108 935 звонков.


  2. На чем написан сервис?
    Сервис состоит из большого количества частей, каждая из которых может быть написана на своем языке. Если кратко, то код CoMagic на javascript, большая часть серверных скриптов на питоне.
  3. Антиспам\Антибот
    В настоящий момент в работе используется несколько моделей: простая эвристическая модель, можель для роботов Яндекса (паттерн яндера), модель по метрике. Планируем добавить модель по кликстриму.
  4. По организации резервного канала (медь)
    Предлагаю организовать на следующей неделе конфкол, подключу инженера связи с нашей стороны для обсуждения задачи, есть уточняющие вопросы.
  5. По кросс-девайсной и кросс-доменной идентификации посетителя
    В основе решения будут использоваться цифровые паспорта Smartis. Smartis знает связь cookies-контакты более 10 млн. устройств Рунета. Сама механика еще в разработке. По данной теме вернемся к вам, когда решение и его описание будет окончательно сформировано.
