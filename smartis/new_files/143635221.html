У сервиса Авито есть возможность написать сообщение автору объявления. 




Смартис виджеты умеет интегрироваться с Авито мессенджером. 

Обращение посетителя из Авито мессенджер будет отображаться в рабочем месте оператора (РМО). 


Инструкция по подключения интеграции с Авито:

ВНИМАНИЕ! Подключать должен тот, кто имеет доступ к аккаунту в Авито - в шаге №6 будет происходит oAUTH авторизация!! (когда открывается окно, в котором надо подтвердить разрешение)


1) Нужны ключи доступа, чтобы отправлять и принимать сообщения Авито
2) На момент 18.02.2022 чтобы получить ключи - надо купить хотя бы базовый тариф (800р 1 месяц) https://www.avito.ru/professionals/tariff
3) Для создания приложения нужно перейти по ссылке https://developers.avito.ru/applications


4) При создании приложения в Авито нужно вставить сформированную ссылку из кабинета клиента


5) После того как вам одобрили ваше приложение со стороны Авито (проверка проходит 3-5 дней) и у вас появились ключи доступа, надо зайти в настройки интеграции Avito и ввести свои 2 значения client_id и client_secret, выбрать подключение. ( нажать сохранить )


6) Далее нужно перейти по ссылке Авторизоваться, и нажать Продолжить ( ссылка сформируется автоматически после нажатия сохранить)


7) Вас перенаправит со страницы Авито на страницу, которая возьмёт все необходимые параметры и можно начинать тестировать отправку и приём сообщений
8) Предварительно нужно иметь на Авито хотя бы 1 товар, чтобы проверить отправку и приём сообщений.

9) Включаем саму интеграцию 


Далее, нужно создать кастомное приложение на стороне Blinger:

заходим в сам Blinger → раздел "Приложения" → нажимаем "Добавить" → находим Custom App, нажимаем добавить



Заполняем имя;

заполняем Custom App ID - придумываем сами; 

webhook url - берем из настроек СКБ (пункт №6 в документации Настройка чатов и мессенджеров)


нажимаем добавить




Создаётся кастомное приложение.

После этого необходимо зайти в редактирование кастомного приложения, для этого находим созданное кастомное приложение и нажимаем "изменить"



Берем отсюда следующие параметры:

-Токен

-Custom Channel Id

-Webhook url to Blinger


и вставляем их в настройки Чаты и соцсети, в то подключение, на которое создавали кастомное приложение, сохраняем. 


Интеграция настроена! 


Если по каким-то причинам интеграция перестала работать, то необходимо её переподключать в следующем порядке:

  1. Переавторизоваться в настройках интеграции Авито в СКБ 

      2. Переподключить блингер полностью блингер

Только в такой последовательности, чтобы всё заработало!



Если приходит такое сообщение - то это означает то, что пользователь, который написал сообщение - забанен (заблокирован) со стороны Авито.
