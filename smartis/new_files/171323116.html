Ручной импорт нужен, чтобы точечно внести значения для импортируемого показателя. Например, вам нужно внести расход по Наружной рекламе за прошедший месяц, при этом данные по другим площадкам вы не вносите. Также ручной ввод пригодится, если вам нужно точечно изменить данные в 1 импортируемом показателе (например, некорректно импортирован расход по 1 площадке - чтобы не перезагружать весь медиаплан, можно через ручной импорт указать корректное число.

Ручной импорт данных находится в разделе Импорт → Занесение данных. Ссылка для облака: https://my.smartis.bi/import


Чтобы выбрать ручной импорт, в окне "Что загружаем" выбираете "Ручной ввод данных"



Шаги выполнения ручного импорта:

  1. Выбрать сайт, для которого нужно загрузить данные, из выпадающего списка "Сайт"

  2. Вы брать период, в который нужно загрузить данные (можно выбрать любой)
  3. Выбрать канал или площадку. Данные можно загрузить:
    - в площадку
    - в канал (без разбивки на площадки. Например, выбрать Канал "Наружная реклама"
    - в канал Весь трафик (тогда данные загрузятся в этот канал, без разбивки на каналы и площадки)

  4. Выбрать, за что оплата (какие данные нужно внести):
    1. Период - в таком случае данные вносятся за период по показателям: Бюджет план, Бюджет факт, Количество обращений план. Стоимость обращений план - будет рассчитан автоматически.

    2. СРА - в таком случае данные за выбранный период вносятся по показателям: Стоимость обращений, Кол-во обращений план. Бюджет план и факт будут рассчитаны автоматически

    3. СРС - в таком случае будут внесены данные о стоимости клика за выбранный период по показателям: Стоимость клика, Количество кликов (план), 

    5. 5. После выбора нужных показателей, ввести данные напротив каждого показателя в указанных единицах.

      6. Нажать "Загрузить данные" 
      1. Если импорт прошел успешно, в нижней части окна появится надпись "Успешно импортировано". Если будут ошибки - они отобразятся также внизу страницы.

      2. Если в площадке/канале, в который вы загружаете данные, ранее уже были данные по выбранным метрикам - появится окно, которое укажет что данные уже есть:

Нужно выбрать, что делать - можно удалить старые данные и сохранить новые, остановить импорт или добавить данные к имеющимся.
в 1 случае данные перезапишутся (заменятся на новые), в случае остановки импорта импорт будет отменен, в случае добавления - новые данные суммируются с ранее загруженными.
