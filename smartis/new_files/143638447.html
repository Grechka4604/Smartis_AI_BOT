Создание интеграции между Blinger.io и Смартис.Виджетами


  1. Если клиент не заведен в Smartcallback, то зарегистрируйте его

  2. Регистрируем аккаунт в Блингере по ссылке https://app.blinger.io/site/registration?smartis=1 (необходимо реальную почту для подтверждения, телефон можно указать вымышленный) 

  3. Заносим данные нового аккаунта блингера сюда в эту доку -  а так же добавляем название клиента для еженедельной проверки в эту задачу - https://app.asana.com/0/1200497059658289/1200497063299421

  4. После регистрации скинуть название аккаунта в чат с блингером (в скайпе - https://join.skype.com/mGU5bn7eDc6E )  чтобы они активировали его

  5. В кабинете Блингер добавляем интеграцию SmartCallback: 





             6. Вебхуку для интеграции берем в ЛК Smartcallback : 


Для этого заходим в ЛК Smartcallback в раздел "Чаты и соцсети" и у Подключения жмем шестеренку: 



7. После создания интеграции сгенерируется appkey, которые необходимо передавать параметром в iframe, для этого копируем сгенерированный ключ 

 



и вставляем его в кабинет Smartcallback : 

            


       8. Чтобы корректно приходили и формировались заявки и переписка в кабинете Smartcallback обязательно настраиваем вебхуки!!! 

Заходим по ссылке https://app.blinger.io/webhooks/index, нажимаем create webhook, вставляем ссылку вебхука из smartcallback (из пункта №4), ставим везде галки, нажимаем create webhook:
