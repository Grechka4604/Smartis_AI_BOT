У нас есть возможность подключить сервис SCB через скрипт CoMagic.

Менеджер Smartcallback заводит нового клиента в системе, забирает токен из скрипта:

далее отправляем менеджеру Comagic/UIS токен скрипта Smartis для активации виджетов в ЛК Comagic/UIS  -  В ЛК Comagic : Сайты  - Общие настройки сайта  - Токен Smartis 

***Размещение токена доступно только с флагом godmode  - https://app.comagic.ru/?godmode *

Таким образом мы подключаем сервис СКБ через скрипт CoMagic. Дополнительная установка скрипта СКБ не требуется.
