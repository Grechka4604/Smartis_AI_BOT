До передачи клиента вы должны убедиться, что у вас:

  1. Зафиксированы в Asana все задачи по клиенту: текущие, предстоящие, на разработке.

  2. Сформирована актуальная таблица контактов для взаимодействия 

  3. Описан процесс подготовки отчётов и иных приложений к выставлению счетов клиенту

  4. Имеются все необходимые материалы для ведения клиента: роадмап (если есть), таблицы, картинки к виджетам, статистика, иные договорённости (например, по счетам).


Передача клиента:

  1. Выделите время для встречи с аккаунт-менеджером 

  2. Передайте задачи по клиенту

  3. Передайте актуальную таблицу контактов

  4. Передайте процесс формирования отчётов к счетам: покажите как сделать отчёт.

  5. Передайте все необходимые материалы для ведения клиента, оповестите об индивидуальных договорённостях, о специфике клиента.

  6. Расскажите с кем коммуницировать и по каким вопросам.

  7. Оповестите клиента о смене аккаунт-менеджера / передайте контакты для связи.
