Что такое селектор?

Это указатель элемента в HTML разметке

Из чего состоит селектор?

Селектор формируется с помощью тегов, атрибутов  и значений атрибутов

Что такое тег и атрибут?

Теги используются для разграничения начала и конца элементов в разметке, к тегам например можно отнести input,form,div,button и тд

Атрибуты это дополнительные значения, которые настраивают элементы, они бывают стандартные (Например class,id,name,type) и кастомные (Например data-attribute-example)

На картинке указаны разные типы тегов и атрибутов : 

Значения которые принимают атрибуты: 

Атрибут id принимает в себя одно уникальное значение (значение будет уникальным на всей разметке страницы). В селекторе id обозначается через знак хештега # (Пример: #id1)

Атрибут class может принимать в себя несколько значений и в селекторе обозначается через знак “точки”(Пример: .class1 ).При наличии нескольких значений в классе внутри селектора из соединяют таким образом (Пример: .class1.class2.class3)

Остальные типы атрибутов в селекторе указываются в квадратных скобках [name=user_phone]

4.1 Как сформировать селектор для значения привязки поля из одного элемента 

Нажмите правой кнопки на необходимый элемент и выберите пункт "Показать код". И у вас откроется панель разработчика с выбранным html-тегом элемента.

Далее нажимайте на этот элемент правой кнопкой и выбери пункт скопировать селектор. 

И у вас в буфер обмена скопируется селектор (у вас он может отличаться): #__layout > div > main > section > section.card-detail._inverted > div.container.v5-container > div.order-controls > div > button:nth-child(1)

Это первый и самый легкий способ формирования селектора для привязки формы

На примере с данной картинки

button[type="button"]  - где button – это тег элемента, type – атрибут элемента, text – значение атрибута

4.2 Как сформировать более уникальный селектор во избежание конфликтов (Используй, если 1 вариант не сработал) 

  1. Необходимо найти селектор родительского элемента div.order-controls
  2. Необходимо найти селектор дочернего элемента #button_query
  3. Соединяем оба найденных селектора в один через пробел : div.order-controls #button_query и вставляем данное значение как селектор элемента "Кнопка отправки формы"
