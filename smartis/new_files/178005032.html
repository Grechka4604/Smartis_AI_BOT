Подключение к коробке

  1. Запросить у Антона/Валеры/Игоря
    1. Доверенный сертификат и пароль от него
    2. Пароль к VPN
    3. Логин и пароль к RDP
  2. Установка VPN
    1. Получить доверенный сертификат и пароль к нему (у Антона/Валеры/Игоря)
    2. Активировать сертификат
      1. Для Windows - прикреплена инструкция в комментариях
      2. Для MacOs
        1. Два раза нажимаем
        2. Вводим пароль от своего пользователя
        3. Вводим пароль от серитификата
    3. Необходимо скачать Cisco AnyConnect
      1. Откройте браузер и скачайте с access.pik.ru установочный файл VPN клиента (Обратите внимание на издателя сертификата. Издатель сертификата должен быть “ICA01PIK”. Если на вашем компьютере установлено несколько личных сертификатов, вам необходимо будет выбрать тот, который вы создавали. Для этого нажмите на кнопку  “Больше вариантов” (“More choices”) и выберите из списка нужный сертификат).
    4. Устанавливаем Cisco AnyConnect
  3. Подключение к VPN
    1. Открываем Cisco AnyConnect
    2. Указываем путь - access.pik.ru
    3. Открывается окно авторизации с логином из сертификата и вводим пароль от него
  4. Подключение к RDP
    1. Открываем приложение для подключения к удаленному рабочему столу (Remina для Ubuntu, Microsoft Remote Desktop для MacOs, Подключение к удаленному рабочему столку для Windows)
    2. Указываем хост - cl01dsk2.dsk2.picompany.ru
    3. При попытке подключения вводим выданный пароль
  5. Подключение к серверу
    1. На удаленном рабочем столе открываете putty
    2. И выбираете уже настроенное подключение
      1. Если его нет, то взять доступы у Антона/Валеры/Игоря
    3. Кабинет Smartis доступен по URL - http://10.97.101.54/
