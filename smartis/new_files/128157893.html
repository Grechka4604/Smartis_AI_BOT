1. Заходим на рабочий стол CMP Project test в Jira


2. Открывается рабочее место сотрудника 



На рабочем столе сотрудника представлены 4 блока: 

  • "просроченные задачи" - это открытые задачи по которым просрочен дедлайн; 
  • "сегодняшние задачи моих клиентов" - это открытые задачи с дедлайном сегодня; 
  • "задач на 7 дней моих клиентов" - это открытые задачи с дедлайном на ближайшие 7 дней; 
  • "все задачи моих клиентов" - это все открытые задачи по моим клиентам; 

Исходя из блоков перечисленных выше сотрудник может сделать следующие выводы: 

  • мои задачи на сегодня; 
  • просроченные задачи; 
  • задачи на ближайшие 7 дней; 
  • список моих задач;

3. Список клиентов с которыми работает сотрудник мы можем увидеть здесь 

 4. Список задач по клиентам можно просмотреть в конкретных фильтрах 

5. Чтобы внести изменения в профиль клиента необходимо 

  • с рабочего стола сотрудника перейти в карточку клиента; 
  • через редактирование внести необходимы правки; 

6. Добавление задач 

Чтобы добавить задачу необходимо

  • в любом месте Jira нажать на кнопку "Создать" 

  • в открывшемся окне выбрать проект "CMP Project test"; 
  • внести тему/дедлайн/описание задачи; 
  • выбрать клиента и исполнителя, сохранить задачу;
