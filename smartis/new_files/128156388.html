Клиенты CoMagic Premium активно работают с  подрядчиками - рекламными агентствами. 

Агентства имеют сильное влияние на используемую для клиента аналитику. 


Обязанности команды Премиум

  • CS менеджеры и аккаунты считают сотрудников агентств клиента, как одни из основных пользователей сервиса. 
    • Контролируем актуальные контакты агентств в карточке клиента в JIRA и структуре
  • Проводим с агентствами сессии обратной связи - раз в месяц, как с основными пользователями. 
    • Контролируем регулярность общения через Карту здоровья клиента
  • NEW Необходимо исключить ситуацию, когда с руководителями агенства общаются  разные CS, каждый по своим клиентам. 

Согласование с Женей Николаевой, CoMagic поток

  • Команда клиента Premium согласует с агентством, что мы поможем с задачами по прямому клиенту и будем получать обратную связь. Менеджеры CoMagic знают об этом. 
  • Полученная обратная связь по нашему сервису должна дублироваться в CoMagic, Жене Николаевой



Планы по развитию модели работы с агентствами: 

  • в Премиум есть человек, отвечающий за привлечение клиентов через рекламные агентства. 
  • CS менеджер инициирует перевод клиентов агентства на Comagic с конкурентов
  • Кураторы из Премиума регулярно общаются с ТОП ЛПР агентств
