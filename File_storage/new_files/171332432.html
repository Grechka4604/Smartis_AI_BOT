Ручной импорт

  1. Импорт данных → занесение данных

  2. В окне "Что загружаем" выбрать "Ручной ввод данных"

  3. Выбрать сайт, для которого нужно загрузить данные, из выпадающего списка "Сайт"

  4. Выбрать период, в который нужно загрузить данные (можно выбрать любой)
  5. Выбрать канал или площадку
  6. После выбора нужных показателей, ввести данные напротив каждого показателя в указанных единицах
  7. Нажать "Загрузить данные" 
Если импорт прошел успешно, в нижней части окна появится надпись "Успешно импортировано"
  1. Попробовать повторно загрузить в площадку\канал данные по выбранным метрикам
Появится окно, которое укажет что данные уже есть
  1. В появившемся окне выбрать "удалить старые данные и сохранить новые"
В нижней части страницы появится сообщение о том, какие данные изменились
  1. В появившемся окне выбрать "добавить данные к имеющимся"
Данные перезапишутся (заменятся на новые)

Импорт CRM-данных

  1. Подготовить xls файл, Заголовки должны быть в 1 строке
    Пример: 

    Тип сущности

    ЖК

    Плановое количество, шт

    ...

    ...

    Договор бронированияЮжная Битца7......
    Договор купли продажи............ 


    2. Заполняем форму: Выбираем из списка, нужную интеграцию → Указываем Тип CRM события. (Можно выбрать определить автоматически) → Загружаем подготовленный xls файл → Нажимаем кнопку "Загрузить данные"

Отобразится подготовка данных из файла. При сверке данные совпадают
  1. Подтвердить сохранение и указать наименование шаблона импорта


Успешно сохранено

Импорт медиапланов

1.  Зайти в раздел “Импорт данных”

2. Выбрать в пункте “Что загружаем?” Медиаплан → ввести для какого сайта

3. Выбрать “Период"

4. Выбрать загружаемый файл по шаблону

5. Нажать “Загрузить”

Появится предварительный просмотр данных из медиаплана, данные сходятся
  1. Проставить вручную соответствие столбцов

  2. Нажать «Импортировать данные»

  3. Проставить соответствие каналам и площадкам

  4. Нажать "подтвердить"
Данные импортированы (проверка в следующем шаге)

1. Зайти в раздел «Конструктор отчетов»

2.  Выбрать сайт, по которому загружались данные

3. Выбрать показатель «Бюджет медиапланов (факт)»  и группировку «Канал» или «Площадка»

4. Выбрать период, за который были загружены данные

5. Выбрать тип «Таблица» и построить

Занесенные данные корректно отображаются в отчете
