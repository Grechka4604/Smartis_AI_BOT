При создании условий Smartis и других привязок используются регулярные выражения.

  • "?" - Это экранирование спецсимвола
  • "_" означает "любой символ", а "\_" означает символ "_"
  • "%" означает "любое количество любых символов", а "\%" означает символ "%"
  • также можно делать экранирование с помощью символов [] - напрмиер, если вам нужно, чтобы символ не считывался как условие регулярного выражения, а воспринимался как символ - нужно его заключить в квадратные скобки
  • ^(79.*)$ - пример регулярного выражения, чтобы выделить номера в формате 79.......
  • для того чтобы прописать условие "и" - .*пример.*.*пример1.*



    Также можно генерировать условия регулярки под вашу задач с помощью Яндекс GPT https://ya.ru/ai/gpt-3 или ChatGPT https://chatgpt.com/ (он только в VPN работает).
    Примеры промптов:
  • Отфильтровать данные, которые содержат какой-то текст, и не содержат какой-то текст: "Мне нужно регулярное выражение, которое выделяет все строки, содержащие слово 'dog' и не содержащие слово 'cat'. Помоги составить такое выражение."

  • Найти все email-адреса в тексте: Пример: "Помоги составить регулярное выражение для поиска всех email-адресов в тексте."

  • Извлечь номера телефонов в международном формате: Пример: "Какое регулярное выражение нужно использовать для извлечения номеров телефонов в международном формате (начиная с +)?"

  • Выделить все даты в формате DD/MM/YYYY: Пример: "Мне нужно регулярное выражение для выделения всех дат в формате DD/MM/YYYY в документе."

  • Найти все слова, начинающиеся с заглавной буквы: Пример: "Помоги найти регулярное выражение, чтобы выделить все слова, начинающиеся с заглавной буквы в тексте."

  • Извлечь все ссылки (URL) из текста: Пример: "Какое регулярное выражение использовать для извлечения всех ссылок (URL) из текста?"

  • Найти все числа с десятичной дробью: Пример: "Помоги составить регулярное выражение для поиска всех чисел с десятичной дробью в тексте."

  • Извлечь IP-адреса из текста: Пример: "Какое регулярное выражение использовать для извлечения всех IP-адресов из текста?"

  • Найти все слова, содержащие определенную подстроку: Пример: "Помоги найти регулярное выражение, чтобы выделить все слова, содержащие подстроку 'abc'."

  • Извлечь все хэштеги из текста: Пример: "Как составить регулярное выражение для поиска всех хэштегов в тексте?"

  • Найти все строки, содержащие только цифры: Пример: "Мне нужно регулярное выражение, чтобы найти все строки, содержащие только цифры."



Если в смартис регулярка от нейросети не работает или работает не корректно, пишите "Такой вариант не работает, напиши еще варианты регулярного выражения, которые решат мою задачу".
