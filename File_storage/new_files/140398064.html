  6. Памятка по заведение задач в Jira 
    Пространство - DOU 
    Наименование задачи - Подготовить документ основание для клиента (юридическое наименование)
    Текст задачи: 

    ИНН клиента - 9705138170

    ID карточки клиента в Инфопин PP и № и дата письменного договора - ID 310749 SMTP №139 от 03.03.2021 (SCB/Аналитика/договор технической поддержки SMTP)

    Коллеги, прошу подготовить ДС для клиента в связи с переходом ООО "Смартис" на УСН с 01.01.2020 по клиенту и контракту указанному выше. 

    Далее возможны варианты: 

    стоимость услуг для клиента не меняется применяем универсальный шаблон (вариант А универсальный шаблон ДС/индивидуальная спецификация - прописать один из вариантов, согласованный с клиентом в приоритете доп. соглашение). 

    стоимость услуг для клиента уменьшается на 20% (вариант В) из спецификации просьба удалить блок за внедрение.

    Окончание задачи: 

    Сформированное ДС необходимо выложить в формате Word в задаче для согласования с клиентом. 

    Первым подписывает - клиент/ООО "Смартис" 

    Способ отправки - почта (адрес) / курьер (адрес + телефон + контактное лицо) 

    Порядок согласования задач в Jira: 

    - сформировать задачу; 

    - поставить испольнителем  запросить подтверждение, что всё оформлено корректно; 

    -  передаёт задачу на согласование в ДОУ; 

    - отслеживать статус задачи в Jira, включаться при необходимости; 

    - ссылка на задачу добавляется в задачу в Asane.
