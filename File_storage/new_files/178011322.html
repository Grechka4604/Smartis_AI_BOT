Доступ к БД

В коробке в домашней папке лежит compose файл для запуска контейнера adminer docker-compose.adminer.yml. Если контейнер не запущен, можно запустить его (например, docker compose -f docker-compose.adminer.yml up -d --build). Adminer будет доступен по порту 8082. С хостовой машины Window можно пробросить порт 8082 через тунель (например, с помощью putty). Тогда adminer будет доступен в браузере по адресу localhost:8082.
