Какую проблему этого человека/людей решаем?:

У клиента есть 2 типа продаж - Розница и корпоративные продажи. На данный момент все пользователи в клиенте видят аналитику только листам Розницы.

РЛ корпоративных продаж не поступают, т.к. для они могут быть переданы в смартис с помощью отдельного метода, который пока не изучен и не реализован.


Для чего делали:

Для отслеживания аналитики по отделу "корпоративные продажи" (продажа для юр лиц)


Что сделали:

отдельный метод для РЛ корпоративных продаж (доступы к методу клиента)

Данные получаем по интеграции с AUTO CRM


В смартис мы получаем в данном методе

  • Рабочие листы 
  • Контракты - аналог договора купли-продажи, но когда еще не выдали автомобиль
  • Потребности КП- это часть РЛ, но для отчетности ее не используют


Как эти данные выглядят в отчетах и CJM

к этим сущностям применяются все те же группировки, которые применимы к другим РЛ. 


Сейчас поля в данных сущностях называются на англ. список полей можно посмотреть в детализации или карточке CRM.
