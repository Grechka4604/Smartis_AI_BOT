Подключение к коробке

  2. Подключение по SSH - ssh root@45.139.25.134 -p 22 (пароль запросить у Валеры/Игоря/Антона)
  3. Коробка доступна по URL - https://smartis.best-novostroy.ru/login

Самая простая в подключении коробка. Можно подключаться прямо со своего рабочего компьютера. Для доступа к БД со своего компьютера можно просто пробросить порты, например, через Putty и далее использовать подключение localhost в своей IDE
