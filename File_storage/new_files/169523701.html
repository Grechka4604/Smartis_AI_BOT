Автоматизация форм: 

1. В родительском элементе формы нужно добавить атрибут "data-scb-form" (ВНИМАНИЕ! наименование формы должно быть уникальным для каждой формы)

2. В подвязываем полях формы необходимо добавить атрибут "data-scb-form-type" который будет определять тип подвязываемого поля и атрибут "data-scb-form-field" который является уникальным идентификатором поля формы

2.1 Атрибут "data-scb-form-type" должен принимать один из значений:

scb-form-text-field -> Текстовое поле
scb-form-select-field -> Выпадающий список
scb-form-phone-field -> Поле с телефоном
scb-form-submit -> Кнопка отправки формы
scb-form-email-field -> Поле c e-mail
scb-form-phone-or-email-field -> Телефон или e-mail
scb-form-convenient-time-field -> Выбор удобного времени
scb-form-offices-field -> Выбор офиса/дилера

2.2 Атрибут "data-scb-form-field" должен принимать уникальное значение из латинский букв и цифр (Например: textFieldUniqueId3213 )

3. Примеры автоматизации форм:
