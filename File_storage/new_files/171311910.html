Пиксели могут создавать как xrm юзеры, так и пользователи от клиента, с другим уровнем доступов. Для того, чтобы пользователи клиента могли создавать пиксели, нужно для отдела, в котором состоят пользователи, открыть доступ к разделу "Пиксели Post-view".
Для этого нужно:

  • перейти в раздел Пользователи и отделы
  • выбрать нужный отдел
  • перейти на вкладку Права доступа
  • в списке выбрать "Пиксели Post-view" 
  • указать уровень доступа Редактирование или Полный доступ


Добавление новых пикселей через интерфейс

  • Все метки заносятся в файл, с каждым клиентом в свой файл https://docs.google.com/spreadsheets/d/1QHfqVd-phATvuULJMqhCQQa1cvJIidmM/edit?usp=sharing&ouid=115792326338049575481&rtpof=true&sd=true*Обязательно проверить ссылку на картинку,чтобы она открывалась. 

  • Далее агентство или клиент пишет Smartis запрос на название площадки. 

  • Smartis выделяет существующую площадку или создает новую.

  • Агентство или мы сами создаем пиксель через личный кабинет Smartis (облако). 

Создание

Нажав на кнопку "Создать пиксель", откроется модальное окно создания пикселя. Обязательные для заполнения поля помечены знаком '*'.

При создании пикселя, автоматически, будут созданы Объявление по полям: "Заголовок объявления", "Наименование рекламной кампании", "Ссылка на файл картинки",  "Полный адрес посадочной страницы куда ведет баннер" - из которого будут взяты 'utm_source', 'utm_medium', 'utm_campaign', 'utm_content', Компания по полю: "Наименование рекламной кампании" и Ключевое слово по полю "Наименование формата размещения".


Когда создаем пиксель, у которого Заголовок объявления такой же, как у ранее созданных пикселей, в поле Наименование рекламной кампании можно редактировать значение и также ввести туда новое (то есть поле Наименование рекламной кампании пустое и редактируемое)

Ссылка на файл картинки - не обязательное поле, можно создать пиксель и без изображения. Однако если нужно добавить картинку, учитывайте:

На картинку действуют ограничения по форматам, можно загружать только jpg,jpeg,png,gif и по размеру не более 10Мб




  • Агентство отдает пиксель на площадку, контролирует корректность установки пикселя.

Таблица

На странице отображается список всех пикселей конкретного клиента

Сортировка

Все заголовки, столбцов, на странице являются кликабельными, при нажатии произойдет сортировка, по этому столбцу, одно нажатие - по возрастанию, второе нажатие - по убыванию. По умолчанию, пиксели отсортированы по столбцу "ID" в порядке убывания. Значения столбцов "Адрес посадочной страницы", "Пиксель на показ" и "Пиксель на клик" являются ссылками. Если надо скопировать ссылку без перехода на страницу, можно нажать правой кнопкой мыши и выбрать "Копировать адрес ссылки"

Фильтрация

С помощью поля "Поиск по пикселям" можно фильтровать таблицу. Столбцы участвующие в фильтрации - "ID", "Кампания", "Объявление".

Выгрузка

Выгрузить таблицу в excel можно кнопкой "Экспорт", в правом верхнем углу.

В экспортируемый файл попадают все поля из таблицы, небольшими отличиями, в значениях полей "Пиксель на показ" и "Пиксель на клик" будут ссылки вместо подписей "Пиксель на показ" и "Пиксель на клик", а объявление будет состоять только из заголовка объявления, без картинки и описания
