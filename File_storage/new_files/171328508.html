ШагиОжидаемый результат
1
  1. Зайти в проекты и сайты (например, клиент 1252)
  2. Выбрать тестовый сайт
  3. Открыть его интеграции
  4. Выбрать одну из интеграций и привязать с логином (например кабинет по ЯД)
Интеграция успешно привязана и попала в верхний список привязанных интеграций
2
  1. Зайти в конструктор отчетов
  2. Выбрать проект тестового сайта
  3. Выбрать показатель соответствующий интеграции (например ЯД, CoMagic звонки или CallTouch звонки)
  4. Выбрать текущий месяц (тут не принципиально, может быть такое что в текущем месяце нет данных - тогда можно выбрать другой период)
  5. Построить отчет
Отчет построен, есть список кампаний
3
  1. Выбрать условие для проверки из списка кампаний в отчете
  2. Вернуться в раздел интеграции
  3. Нажать на шестеренку в привязанной интеграции
  4. Ввести часть условия в поле "Наименование кампании содержит" в формате .*test.*
  5. Сохранить
  6. Еще раз построить тот же отчет
В построенном заново аналогичном отчете должны отобразиться только кампании, соответствующие введенному условию
4
  1.  Пройти в раздел "Проекты и сайты", выбрать нужный сайт и зайти в его настройки

  2. В доступных для привязки интеграциях необходимо выбрать интеграцию Коннектор Smartis.CRM и привязать ее к сайту

  3. Кликнуть на иконку шестеренки на плашке интеграции с коннектором

  4. Выбрать настройки интеграции
    (Примеры: Название объекта содержит + Привязка к полю сущности CRM
    Выбор конкретных наименований объектов + Привязка к полю сущности CRM
    Если заполнить одновременно Название объекта содержит и Выбор конкретных наименований объектов, то будет использоваться только 1 из этих условий, поэтому вариант, когда вы заполняете эти 2 поля одновременно, использовать не нужно.

  5. Построить отчет по данной интеграции

При выборе условий привязки они применяются и работают корректно - то есть данные в отчете показывает с учетом настроенных условий привязки

5
  1. Зайти в настройки сайта → настройки привязок crm-аккаунта

В настройках при выборе условий в регулярных выражениях должен быть обычный input с вводом строки

6
  1. В настройках интеграции привязать только по полю сущности CRM
  2. Построить отчет
  3. (как пример)
В отчете должны отобразиться только те данные у которых в выбранном доп поле CRM указано соответствующее название (например, в отчете по договорам должны отобразиться только такие где в поле содержится название - как на скрине - Айвазовский)
