Цель: систематически приносить пользу клиенту/ам по инициативе CoMagic (проактивная работа с клиентом).  

Участники процесса: клиент, аккаунт-менеджер, cs-менеджер, руководитель отдела, 

Инициатор: аккаунт-менеджер/cs-менеджер

Инфо поле: Jira

KPI процесса: 

KPI - число выполненных задач по проактивной проработке клиентов.  

Как контролируем: контролируем результат выполнения задач в Jira; 

Результат: результатом выполнения процесса будет являться  то, что клиент подтверждает получение ценности от выполнения проактивной задачи аккаунт-менеджером, CS-менеджером

Владелец процесса: руководитель отдела

Статус процесса: не работает.

 

Краткое содержание процесса.


  1. Каждый понедельник сотрудник ставит себе задачу в Jira, с темой "Проактивная проработка клиентов" минимум по 1 клиенту или более.
  2. Чтобы выбрать клиента сотрудник проверяет личный кабинет, начиная прорабатывать клиентскую базу с организаций, которые имеют наименьшее количество обращений. 
  3. Проверяя личный кабинет сотрудник по следующим параметрам: 
    - количество потерянных/причины; 
    - наличие интеграций; 
    - рекламные компании (на момент не актуальных); 
    - проекты (сайты); 
    - наличие номеров, которые можно отключить; 
    - охват; 
  4. На основании проведенной проверки готовит рекомендации, которые вписывает в соответствующую задачу в Jira, в комментарии; 
  5. Далее на основании подготовленной информации совершает исходящий звонок клиенту, во время которого прорабатывает с клиентом сформированный набор рекомендаций. 
  6. По итогам выполненной задачи сотрудник вносит комментарий в задачу и закрывает её. 

    Дополнительные темы для коммуникаций: 
    - приглашение на мероприятие/обратная связь по мероприятию; 
    - новости о сервисе; 
    - фидбек по задачам/сервису; 
    - фидбек по хотелкам; 
    - масштабное предстоящее обновление продукта (обновление ЛК, системы отчетов и пр.); 
    - комплексное решение для отрасли недвижимости CoMagic.Realty, которое позволяет повышать конверсию по всей воронке продаж;
    - оптимизация личного кабинета;
