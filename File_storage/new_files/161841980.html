Документ описывает обозначения, а так же определяет их использование при описании бизнес-процессов в юните Смартис. 

ЭлементИзображениеРасшифровка (интерпретация)
Стартовое событие (пустое)

Стартовое событие указывает на то, в какой точке берёт начало процесс
Стартовое событие сообщения

Стартовое событие, в котором процесс начинается с определённого сообщения
Стартовое событие таймера

Стартовое событие происходящие по расписанию (в определённые моменты или периоды времени)

Промежуточное событие

Окончание процесса
Событие окончания сообщением

Окончание процесса с сообщением
Задача службы

Задача выполняема автоматизированным ПО
Задача пользователя

Задача выполняемая сотрудником. Так же в ходе выполнения задачи сотрудник может использовать различное ПО
Задача (абстрактная)

Задача-единица работы.
Эксклюзивные шлюзы (строгое "или")

Логический оператор, при ветвлении направляет поток лишь по одной исходящей ветке.
Параллельные шлюзы (развилка "и")

Логический оператор, при разделении потоков все параллельные ветки активируются одновременно. При завершении операций по разным потокам оператор ждёт завершении всех параллельных веток и затем активирует исходящий поток.
Поток управления (без условный)

Определяет порядок выполнения действий
Поток сообщений

Описывает информационный поток между участниками
Объект данных

Объект данных представляет собой информацию, которая обрабатывается в ходе процесса, например, документ или письмо
Аннотация

Второй стандартный Артефакт нотации BPMN - представляет собой механизм, при помощи которого разработчик модели может добавлять на диаграмму дополнительную информацию, являющуюся важной для конечного пользователя диаграммы. Текстовые аннотации используются для уточнения значения элементов диаграммы (добавления комментариев, пояснений и другой текстовой информации) и повышения её информативности и лёгкость для понимания любым бизнес-пользователем
Хранилище данных

Хранилище данных - объект, который процесс может использовать для записи и извлечения данных, например, база данных.
