Обозначения

Этап проекта/жизненный цикл проекта – это этапы, которые проходит проект с момента создания до завершения.

Выделяют следующие этапы проекта: 

Инициация - обсуждение необходимости, целесообразности проекта. 

Планирование - формирование плана работ по реализации проекта, оценка рисков и затрат. 

Исполнение/в работе - выполнение работ по проекту. 

Завершение - завершение проекта, фиксация подтверждения о выполненных работах, достигнутого результата. 


Статус проекта - обозначает текущее состоянию по проекту, отражает на сколько успешно (или неуспешно) сейчас выполняется проект. 

Могут быть следующие статусы: 

По плану - означает, что ход выполнения работ указывает на то, что сдача проекта произойдёт в рамках указанного дедлайна. 

Отстаёт - означает, что мы прогнозируем (ориентируясь на ход работ), что по каким-то причинам (нам известным) проект будет сдан позже установленного дедлайна.

Под угрозой - означает, что есть риск не выполнения проекта или не достижения предполагаемого результата, требуется дополнительное обсуждение с заказчиком. 

Отложено - означает, что проект по согласованию с заказчиком поставлен на паузу. 

Завершено - означает, что проект завершён. Проект может быть как завершён с заданным результатом (положительный исход) так и результат не достигнут.
