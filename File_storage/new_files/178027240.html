Потребность в подписании документов в офисе г. Москва (генеральным директором) может возникнуть в процессе осуществления хозяйственной деятельности.

Подготовленные в рамках задач по процессу → Подготовка внутренних документов (приказов/доверенностей/официальных писем и т. п.) документы аккумулируются специалистом по документообороту в один пакет для организации его передачи на подписание в офис г. Москва.

1. Передача документов на подписание в офис г. Москва (зона ответственности специалиста по документообороту):

Специалист по документообороту отправляет пакет документов в офис г. Москва одним из нижеуказанных способов:

  • путем передачи через Директора департамента сквозной аналитики М.Терехова

  • посредством курьерской службы Major Express

После отправки пакета документов специалист по документообороту в группе Документооборот в Telegram сообщает о передаче пакета документов в офис г. Москва. В случае передачи пакета документов посредством курьерской службы специалист по документообороту сообщает ДАТУ ДОСТАВКИ пакета документов курьерской службой в офис г. Москва. В ДАТУ ДОСТАВКИ главный менеджер по работе с персоналом/помощник менеджера по работе с персоналом должен присутствовать в офисе г. Москва.

2. Прием документов в офисе г. Москва (зона ответственности главного менеджера по работе с персоналом/помощника менеджера по работе с персоналом):

Главный менеджер по работе с персоналом/помощник менеджера по работе с персоналом принимает пакет документов (от Директора департамента сквозной аналитики М. Терехова или от курьерской службы), информирует генерального директора о наличии в офисе г. Москва пакета документов для подписания и согласовывает с ним дату его присутствия в офисе.

3. Подписание пакета документов в офисе г. Москва (зона ответственности главного менеджера по работе с персоналом/помощника менеджера по работе с персоналом):

В день присутствия генерального директора в офисе г. Москва главный менеджер по работе с персоналом/помощник менеджера по работе с персоналом лично передает ему документы на подпись.

4. Отправка пакета документов в офис г. Тула (зона ответственности главного менеджера по работе с персоналом/помощника менеджера по работе с персоналом):

После подписания документов генеральным директором главный менеджер по работе с персоналом/помощник менеджера по работе с персоналом отправляет пакет документов в офис г. Тула любым из способов, указанных в п.1 настоящего регламента, в группе Документооборот в Telegram сообщает о передаче пакета документов в офис г. Тула.

При отправке документов посредством курьерской службы главный менеджер по работе с персоналом/помощник менеджера по работе с персоналом:

  • вызывает курьера по телефонам, указанным на сайте курьерской службы Major Express https://major-express.ru/contacts.aspx
