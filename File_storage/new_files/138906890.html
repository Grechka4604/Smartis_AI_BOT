В нынешнее время 40-50% посетителей посещают сайт с помощью мобильных устройств. 

Для таких посетителей мы можем настроить и включить мобильные акции, для увеличения конверсии из посещения в обращение.

Выглядят мобильные акции следующим образом:



Мобильные акции настраиваются следующим образом: 

нажимаем на нужный домен → нажимаем "Акции"




Открывается страница с настройкой акций. Первым делом надо нажать на "добавить предложение"!

После этого можно приступать к заполнению всех остальных полей.



Далее заполняете поля, пример ниже, нажимаете сохранить. 

Ссылку на предложение можете не заполнять. 



Чтобы поменять цвет в мобильных акциях,



нужно зайти в настройки версии, найти раздел "7. Настройки Mobile", найти пункт:

34. Основной цвет акций
*mobileHotOffersColor

и в этом поле поменять цвет.




Мобильные акции появляются только при добавлении определенных внешних кодов, см. ниже.

Добавляем эти внешние коды в поле "Код при завершении подгрузки виджета"

Вместо числа "40" ставим необходимо количество секунд для открытия. 

Первые два кода отрабатывают 1 раз за сессию. Последний код будет работать всегда. 


// откроет акции через 40 секунд, сбрасывается после обновления страницы (в статистике openType = 255)
setTimeout(function() {
Smartis_Mobile_HotOffers.openByTimeout(1)
}, 40 * 1000);


// откроет акции после закрытия виджета через 40 сек (в статистике openType = 253)
setTimeout(function() {
Smartis_Mobile_HotOffers.openByTimeoutAfterCloseWidget(1);
}, 40 * 1000);


// открывает акции по клику на элемент с селектором #button (в статистике openType = 254)
setTimeout(function() {
Smartis_Mobile_HotOffers.openByClick('#button');
}, 40 * 1000);


//открывает акции через 40 секунд. Нет ограничения по открытию за 1 сессию.  (в статистике openType = 252)

setTimeout(function() {
Smartis_Mobile_HotOffers.openByForce(1)
}, 40 * 1000);


ВАЖНО:
После того, как Вы заменили текст в мобильной акции необходимо не только сохранить изменения в настройках акций, но и очистить кэш в настройках домена! В противном случае изменения не сохранятся.
