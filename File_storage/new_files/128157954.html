Требования к заполнению задачи в Jira. 

Проект - CMPM

Тип Задачи - Задача

Тема - Анализ отчёта номера коллтрекинга

Срок исполнения - [по ситуации]

Описание - 

  • период анализа (с ... по ....)
  • вносим какие проекты/сайты анализируем у клиента; 
  • процент охвата по каждому сайту;
  • процент звонков без сессии; 

например, 

 "период анализа - с 1.06.2020 по 23.06.2020

проект настоящее.москва:

- охват 95,2% 

- звонков без сессии 10,96%" 

или можно вставить скрин


Комментарии

в комментариях прописываем, что сделали

- чтобы исправить ситуацию с охватом; 

- чтобы исправить ситуацию с звонками без сессии;

- в чём была причина, почему у клиента были звонки без сессии; 


Порядок работы с задачей

Проработка задачи происходит по этапам: 

  1. Составить рекомендации
  2. Презентовать клиенту
  3. Согласовать с клиентом реализацию рекомендаций (взять обязательства, кто и когда вносит изменения)
  4. Согласовать допустимую долю и способ измерения (только динамические РК, какие-то Рк считаем, какие-то нет)
  5. Замерить показатели повторно
  6. Подвести итоги (другими словами зафиксировать показатели на которые мы вышли/цель достигнута не достигнута)
  7. Закрываем задачу
