Подключение к коробке

  1. Запросить у Антона/Валеры/Игоря
    1. логин и пароль к RDP (после запроса высылается лично клиентом на вашу рабочую почту)
    2. приложение для двухфакторной авторизации (высылается письмо от клиента на вашу личную почту)
  2. Подключение к удаленному рабочем столу (опционально: внизу прикреплен готовый файл RDP)
    1. В зависимости от ОС, используйте приложение поддерживающее подключение по RDP (MacOS - Microsoft Remote Desktop, Linux - ???, Windows - Подключение к удаленному рабочему столу)
    2. IP-адрес компьютера - RDCB.TEKTA.LOCAL:3389
    3. Шлюз - rdgw.tekta.com
    4. Подключайтесь
    5. Как только запросит авторизацию введите логин и пароль от своей учетной записи (в письме от клиента)
    6. После на приложение для смартфона Multifactor приходит уведомление о попытке входа, нажимайте “ОК”.
  3. Подключение по SSH внутри удаленного рабочего стола
    1. ssh admsmart@10.0.40.58
    2. пароль от ssh получить у Антона/Валеры/Игоря
  4. Приложение доступно внутри удаленного рабочего стола по URL - https://smartis.tekta.com/login

Multifactor

Важно! Для двухфакторной аутентификации после ввода реквизитов rdp в процессе подключения используется Приложение для смартфона Multifactor. При подключении не всегда возникает popup приложения с запросом подтверждения. Так что лучше перед подключением зайти в приложение на смартфоне и в процессе подключения ждать появления кнопки ОК в приложении.

Общая информация

  1. В коробке есть Adminer для работы с MySQL - https://10.0.40.58/GHwlve456433KGwjcqqkwjcn3455212kjn/adminer-4.7.5.php
  3. Все команды с docker-compose запускать в папке /var/smartisstat/
  4. Чтобы обновить ssl сертификат необходимо:
    1. поместить файлы в папку /var/smartisstat/nginx/certs
    2. обновить конфиг в папке /var/smartisstat/nginx/config
    3. перезапустить контейнер nginx
