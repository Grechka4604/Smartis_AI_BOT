Цели

  1. Переход к ноябрю на наш сип - транк, и, как следствие, сокращение затрат на связь
  2. Реализовать для клиента возможность получать от нас все данные о звонке с помощью определенного идентификатора

Критерии успешности/Требования

  • Бесперебойная работа телефонии:
    плавный переход на наш sip-trunk по отдельным проектам
    -успешное тестирование каждого перешедшего на наш sip виртуального номера
    -полный переход на наш sip-trunk по всем проектам клиента
  • Получение данных о звонках по sip-trunk:
    -В целом необходимо передавать:
    номер А(АТС), номер В (номер абонента), call session id, время звонка, продолжительность звонка
    -Передача call session id в поле "divrsion"
  • Экономия клиента на связи
