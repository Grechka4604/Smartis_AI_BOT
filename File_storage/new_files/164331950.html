1. Список сегментов, в которые попала персона


    Если персона попала в один или несколько сегментов, то в детализации (сырых данных) отобразится список сегментов. Также список сегментов, в которых есть персона, можно посомтреть в CJM. Если навести курсор на название сегмента в CJM, то отобразятся даты, когда персона попала в сегмент. Персона может быть записана в сегмент несколько раз (на примере на скрине - персона 2 раза посещала более 2 страниц сайта и не оставила обращение). Важно, что при формировании сегмента для передачи в рекламные кабинеты, мы считаем только уникальных персон.

2. Фильтры по сегментам

Фильтр "Сегмент" - позволяет выбрать всех персон, которые уже находятся в выбранном сегменте, в выбранном периоде

Фильтр "Сегмент на дату попадания в сегмент" - позволяет выбрать всех персон, которые попали в выбранный сегмент в указанном периоде


3. Категория показателей "Показатели CDP"

Категория показателей "Показатели CDP" создается автоматически, когда вы подключаете автоматическое решение (кроме Антифрода), а также когда вы создаете сегменты для кастомных клиентов CDP из конструктора отчетов. Названия показателей = названия сегментов.


4. Антифрод: процент вероятности фрода и детализация % фрода

После подключения Антифрода у каждого обращения появится столбец "Процент фрода" - в значение можно провалиться и посмотреть детализацию

Детализация по проценту фрода: список всех проверенных стратегий, напротив каждой зеленым - 0 или минус проценты, красным - где добавили % вероятности фрода в результате проверки. Расчет ведем от 50% - так как изначально вероятность фрода 50/50. Детализацию никак нельзя выгрузить отедльно - только смотреть в интерфейсе


5. Фильтры для Антифрода

Фрод: процент вероятности - позволяет отфильтровать обращения с выбранным условием.
