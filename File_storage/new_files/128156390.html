План проведения вебинара

  1. Решили проводить мероприятие 
    1. Дата
    2. Тема-Докладчики
  2. Приглашение спикеров
    1. Выбрать компании, которые хотим видеть. Отсортировать от меньшей вероятности к большей
    2. Получить от Родиона задизайненное приглашение для спикеров
    3. Определить срок, когда должны получить подтверждения
    4. Сделать 2-3 волны приглашений
    5. Ежедневный мониторинг прогресса
  3. База для приглашения
    1. Рассылаем email
    2. Указать конкретные контакты, кому звоним
  4. Мониторинг регистраций
    1. Помечаем, сколько действующих клиентов зарегистрировалось
    2. Сводная таблица с числом зарегистрировавшихся из действующих
  5. Приглашение участников
    1. Тренировка Сценария с прослушкой разговоров
      1. Продать основные тезисы, что полезного услышит клиент
      2. Получить подтверждение о добровольном участии (самостоятельно зарегистрировать)
    2. Скорректировать сценарий
    3. Дожимаем в несколько этапов - проверить цепочку писем/напоминалок
    4. Общий с Родионом мониторинг именно по действующим клиентам
  6. Обязательный участие/просмотр записи мероприятия всеми сотрудниками (как минимум, если контент новый)
  7. Обработка после вебинара
    1. Выделить из базы действующих клиентов - просматривает каждый менеджер, выделяет своих
    2. Сценарий звонка
      1. Тренировка
      2. Фокус на обсуждении потребностей клиента
    3. Прослушка выборочная
    4. Перенести новые контакты в JIRA
