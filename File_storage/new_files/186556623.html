Основная информация

ReQuest — это онлайн-конструктор чат-лендингов, позволяющий быстро создавать интерактивные посадочные страницы в формате чата для эффективного привлечения клиентов.

Сервис разработан для повышения конверсии и вовлеченности посетителей по сравнению с традиционными сайтами.

Подключение интеграции

  1. Подключение интеграции в кабинете Smartis.
    1. Нажимаем подключить
    2. В настройках интеграции вводим только ее наименование, url для Webhooks сгенерируется автоматически.
    3. После подключения копируем url.
  2. Настройка в личном кабинете rqchat.ru
    1. В личном кабинете переходим в раздел Чат-лендинги и переходим в нужный сайт.
    2. В выбранном сайте переходим в раздел Интеграции
    3. Находим интеграцию со Smartis и включаем селектор
    4. После чего появится поле, куда мы вставляем url, который скопировали в настрйоках интеграции в Smartis
    5. Все готово! Теперь Smartis будет получать эти данные.
