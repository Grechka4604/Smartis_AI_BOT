В Smartis существуют следующие разделы:

  1. Дашборды - сохраненные построенные отчеты в форме виджетов по заданным метрикам и параметрам. 
  2. Конструктор отчетов - раздел для построения отчетов по метрикам/показателям в различных срезах (по проектам, месяцам, каналам.
  3. База обращений – база данных, в которой фиксируются все обращения из трекинговых сервисов.
  4. CDP- продукт smartis, основывающийся на сегментировании персон.
  5. Импорт данных – позволяет импортировать данные в систему аналитики в заданную для импорта метрику и проект/дату. 
  6. Администрирование – позволяет управлять настройками системы, имеет несколько подразделов:
    • Проекты и сайты: раздел для создания и управления проектами/сайтами, изменения настроек сайтов и привязки данных к проектам/сайтам; 
    • Интеграции: позволяет подключать и настраивать интеграции с основными сервисами, необходимыми для построения сквозной аналитики; 
    • Пользователи: позволяет создавать/удалять пользователей и отделы, задавать и изменять права доступа к данным для них; 
    • Конфигуратор: позволяет настраивать кастомные параметры для анализа данных (группировки, категории, тел. номера не участвующие в анализе).

    • Пиксели Post-view: раздел, позволяющий создавать пиксели и мониторить статистику по созданным пикселям.
