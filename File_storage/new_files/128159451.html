Закрывающие документы, оплата, сроки.Обсудить оптимизацию процесса выставления счетов

У клиента есть боль, касающаяся оплату счетов.

1. в 26 числах мы обычно направляем клиенту примерные списания за месяц, формируем приложения ( 4 приложения);

2. в первых числах месяца - пишем клиенту, сколько в итоге вышло денег - по итогу решаем,какое из приложений будем корректировать;

3. отправляем данные по приложениям и сумма в Виртуоз

4. Виртуоз делает закрывающие документы по Приложениям и отправляет курьером клиенту

5. клиент пересогласовывает то приложение, в котором были изменения ( для него это болезненно)

Результат: Обсудить с ВИРТУОЗ ПЛЮС возможность отсрочки платежа, так как это облегчит всем жизнь.
