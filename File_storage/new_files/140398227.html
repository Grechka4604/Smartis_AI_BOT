Архитектура решения:

Собрать все данные - Склеить - Визуализировать - Поддерживать достоверность - Рекомендации


Собрать - мы поставщик данных.

  • Нет экспертизы поддержки больших данных под нагрузкой.
  • Нет пост вью аналитики с ЦИАН и классифайдами.

Следствия:

  • Будет 6-10 месяцев турбулентности в работе продукта.


Склеить - архитектурное ограничение Елли.

  • Нет правил склейки данных - дорогой процесс, который выстраивался 6 лет в недвижимости на 50 проектах!
  • Мало данных для склейки - Возможность дополнить проект через матчинг
    • Нам не нужно клеить из-за единой архитектуры всех каналов связи

Показать - карточку клиента, модели атрибуции.

Какой процент матчинга сейчас? Какое целевое значение? Какие сроки?

Следствия:

  • Низкий матчинг. Дорого и долго повышать и поддерживать.
  • Нет возможности быстро сравнивать модели атрибуции
  • Нет возможности быстро проверить достоверность


Визуализировать

  • Power BI - тяжелый для создания собственных разрезов данных. Требуется экспертиза внутренняя
  • Высокий риск ошибок, долгая отладка. Из-за высокой гибкости продукт не учитывает логических органичений для аналитики маркетинга и продаж.
  • Умеем отгружать в Power BI подготовленные данные.

Показать - Конструктор отчетов

Следствие

  • Тяжело вовлечь сотрудников/новых пользователей/подрядчиков в работу с данными.


Поддерживать достоверность - стратегический риск.

  • Нет ответственности за ключевой показатель качества сквозной - матчинга
  • Система не контролирует источники данных.
  • Приходится инвестировать при появлении новых решений.
  • В Power BI нет автоматического мониторинга достоверности
  • Нет анализа корректности по загрузке медиапланов

Следствия:

  • Риск ошибки при принятии решений
  • Дополнительные инвестиции в подготовку данных


Рекомендации

Опыт 400 застройщиков, 40 сквозных аналитик для недвижимости.


Условия:

В Елли от 500 до 1 млн рублей в месяц за аналитику, без источников данных.

Наше предложение включает комплекс.
