    1. ИТОГИ ПО ЗАДАЧЕ

      Property Finder:
      - Получили информацию, что номера клиентов не показываются напрямую потому, что они делают подмену номера для скоринга, при этом, есть приложение для Android и iOS, которое называется Lead Tracker и позволяет увидеть все прямые номера.
      - Получили от технической поддержки Property Finder прямое письменное подтверждение того, что размещать Landline номера телефонов - можно.
      - Получили информацию от технической поддержки Property Finder, что если вотсап заругистрирован на Landline номер телефона - сообщение не пройдет. При этом, можно указать отдельный мобильный номер для получения WhatsApp обращений в этом случае.
      - Ранее получили информацию по API документации для вотсапа. В случае получения обращений WhatsApp по API проблема с невозможностью использования городского номера уходит и решать ее не нужно. Мы потенциально сможем сами передать такое сообщение агенту средствами нашей коммуникационной платформы и написать клиенту с любого номера.
      - В целом есть впечатление, что нам не нужны какие-то доработки от Property Finder для того, чтобы начать отследивать коммуникации из этого источника. В идеале нужно получить некий тестовый аккаунт для разработки и тестирования интеграции, а также возможность задавать вопросы, связанные с этой интеграцией.
