Основной фокус и задачи на 2023Цель обсуждения
    1. Определили, что сейчас основной ваш фокус - это настройка сквозной аналитики. Ориентировочно в течение марта должен быть готов функционал со стороны вашего IT-департамента, чтобы начать приостановленное внедрение.

    2. По вопросу нашей интеграции с CallToVisit - уточнила.


      Вы можете получать заявки с формы, как и обсудили на встрече.
      По нажатию на вашу кнопку должен открываться наш виджет, у которого в настройках стоят «Формировать заявку такси», без этого интеграция не будет работать.
      Есть несколько вариантов настроек - вы можете прописать наш скрипт на свою кнопку, или мы сами в настройках можем указать class или id вашей кнопки, по нему будет открываться виджет.
      Далее через наших разработчиков мы подключим вам интеграцию с заданными настройками. Интеграция работает только с заявками с типом такси, оставленными через наш виджет
