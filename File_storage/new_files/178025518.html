Расторжение заключенного с клиентом договора осуществляется путем подписания соглашения о расторжении договора (соглашение сторон).

До постановки задачи в Jira менеджер обязан (зона ответственности менеджера):

- согласовать с клиентом дату окончания предоставления услуг по договору;

- согласовать сумму счета, которая будет выставлена за последний отчетный период оказания услуг по договору (бизнесовая часть).

1. Постановка задачи (зона ответственности менеджера)

  • менеджер формирует задачу в Jira на специалиста по документообороту (Smartis Документооборот (SMA))

Тип задачи: Клиент SMARTIS

Тема: расторжение договора <наименование юр. лица>.

Юридическое наименование клиента (кратко): <наименование юр. Лица>

Тип документа: договорной документ

Описание: дата расторжения, номер договора, согласованная сумма последнего счета, способ подписания документа (ЭДО/курьер*)

* обязательно указываются контактные данные: ФИО и телефон лица для получения документа от курьера.

2. Формирование документа (зона ответственности специалиста по документообороту)

Специалист по документообороту

  • берет задачу в работу (переводит задачу в статус «Подготовка документа»)

  • запрашивает у старшего бухгалтера информацию о наличии/отсутствии задолженности клиента за услуги (при необходимости формируется и направляется клиенту акт сверки взаимных расчетов)

  • формирует документ по соответствующему шаблону (с долгом/без долга), вкладывает его в задачу, тэгает менеджера с комментарием «Отправить клиенту документ на согласование» и переводит задачу в статус «Согласование документа»

  • менеджер отправляет сформированный документ на согласование клиенту


3. Согласование документа (зона ответственности менеджера/ведущего юриста)

  • менеджер организует согласование документа с клиентом*

  • после согласования документа с клиентом менеджер тэгает специалиста по документообороту

* тэгает ведущего юриста (в случае, если клиент не согласовал документ)

4. Подписание документа (зона ответственности специалиста по документообороту)

  • после согласования документа с клиентом специалист по документообороту переводит задачу в статус «Подписание» и организует подписание документа (аналогично процессу → Заключение договора с клиентом)

  • в дату расторжения договора менеджер останавливает предоставление услуг (зона ответственности менеджера). Фактическая дата отключения услуг должна быть равна дате расторжения в документе!

5. Закрытие задачи (зона ответственности специалиста по документообороту)

  • после получения подписанного документа в системе ЭДО/оригинала документа в бумажном виде (в случае, если подписание документа осуществлялось сторонами не посредством ЭДО) специалист по документообороту формирует отчет в задаче, тэгает менеджера с комментарием «Задача выполнена», размещает скан документа в электронном архиве, закрывает задачу


В дату расторжения договора менеджер останавливает предоставление услуг (зона ответственности менеджера).

Фактическая дата отключения услуг должна быть равна дате расторжения в документе!
