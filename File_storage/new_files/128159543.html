Повестка встречи

Сценарий демо:

  • Бренд менеджеру нужна сводка по бренду, который он ведет. Соответствие РК и бренда могут предоставить. 
  • Основной акцент на продажу автомобилей в разбивке бренда, также есть сервис.

  • В идеале нужно, чтобы медиаплан бренд менеджеров отображался в личном кабинете. Готовы дать пример. 
  • Часть каналов с ручным вводом расходов.
  • Офлайн источники рекламы (смски, наружка, пешеходы в ДЦ).

  • По контексту нужен общий срез по всему контексту по всем сайтам. 
  • Доля канала от общего числа обращений по бренду/сайту. 

  • Смотрят данные с нарастающим итогом. Когортный анализ.

  • Сравнивать периоды. 
  • Абсолютные и относительные показатели.

  • Ассоциированные конверсии по количеству обращений по каналу. - не самый горящий запрос.  Окно ретроспективы до полугода. 

  • Расшифровка каждого столбца - как считаются параметры в отчетах.

  • Перетекание между дилерскими центрами - 42 минута записи.
  • Перетекание между продажами и сервисом.

  • Никакого Power BI, хотим в личном кабинете. 
  • Хочу пример готового отчета с цифрами. 


Преимущества перед Колтач и всеми коллтрекингами:

  1. Ключевая экспертиза - У нас есть опыт доведения доли пешеходов/неизвестных до уровня ФСК - 80% матчиться. 75% в среднее, по 20 проектам. Чаще всего приходят 40 - 55% максимум. История Автомира.
  2. У Колтача вряд ли будет разделение по брендам по интегрированным РК. Нельзя отчетность группировать так гибко, на сонове данных в разных системах. 
  3. Только мы можем учитывать в отчетах любые признаки из CRM. Только мы можем группировать трафик без привязки к сайту. Специализированные срезы под компанию, например, по бренду. Мы можем работать с представлениями, как в ГА.
  4. В Колтаче нет План/Факт отчета по медиаплану. (План по лидам, продажам, на любой период)
  5. Только мы умеем сохранять ссылки на конкретный срез с фильтрами и группировками. Удобно скидывать коллегам и подрядчикам готовый отчет.
  6. Есть крутой отчет по контексту. Предустановленный отчет для контекста со всеми показателями - просмотры, визиты, стр и т.д. Какой-то новый. Уточнить у Смартис. 
  7. Когортный анализ в Колтаче только запускается.
  8. Календарь событий - как в Метрике.  Можно помечать, что стало причиной изменений в показателях. Только у нас.
  9. Гибкие права доступа и коробочное решение на стороне клиента. Максимальная безопасность данных из CRM
  10. Полноценная мобильная версия полноценная - только у нас.
  11. Умеем отгружать в Яндекс Аудитории напрямую из ЛК. Только мы.
