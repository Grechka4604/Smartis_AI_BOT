Подключение интеграции 

Интеграция доступна для подключения в разделе интеграция. Принцип итнnеграции с Террасофт очень похож на интеграцию с MicrrosofDynamics. 

Главная особенность Террасофт - возможность настройки большого количества кастомных связей между разными справочниками и сущностями. Это позволяет получать выборки по точечным процессам и контролировать их в рамках системы. 

  1. Зайти в раздел интеграции и выбрать террасофт.

2. Ввести необходимые значения в интеграцию, а именно:

• Создать пользователя в Terrasoft c правами на чтение необходимых для интеграции справочников и полей.
• Предоставить HTTP/HTTPS адрес CRM в формате: htps://crm.company.ru/MAIN

Также, если у клиента настроена конфигурация, можно указать номер этой конфигурации в настройках интеграции, по умолчанию в этом поле указывается значение - "0".

3. Интеграция появится в списке доступных к подключению интеграций

4.Для того, чтобы данные из Амо воспроизводились в Smartis, после подключения интеграции автоматически создастся коннектор Smartis.СRM. Справка по коннекотору: 6. Коннектор Smartis.CRM

Привязка данных к интеграции 

Для того, чтобы данные  отобразились в статистике проекта в Smartis, необходимо привязать интеграцию к нужному проекту. Данные мы всегда привязываем к базовой единице - сайту.


  1. Для привязки необходимо пройти в раздел "Проекты и сайты", выбрать нужный сайт и зайти в его настройки. 
  2. В доступных для привязки интеграциях необходимо выбрать интеграцию и привязать ее к сайту. 


  3. После того, как привязали интеграцию к сайту, никакие дополнительные действия не нужны - интеграция отобразится в разделе "Привязанные к сайту интеграции"

  4. Из самой интеграции  привязки по разным условиям для отдельных сайтов настроить невозможно - эти настройки выполняются в коннекторе Smartis.СRM, который также нужно привязать к сайту (выбрать его из раздела доступных интеграций и нажать "Привязать к сайту"). О том, как настроить привязки к сайт с помощью коннектора - читать тут: 6. Коннектор Smartis.CRM

Синхронизация данных

Синхронизации данных производятся с определенной периодичностью:

• Каждую ночь в 1:00: синхрон всех сущностей
• Каждый час: ежечасный синхрон всех сущностей
• Каждый понедельник в 2:00: еженедельный синхрон всех сущностей 
• Каждый месяц 01 числа в 4:00 : ежемесячный синхрон всех сущностей
• Каждый день в 6:00: ежедневный контроль удаленных за последние 6 месяцев

Набор данных

По умолчанию Smartis не сохраняет все справочники. Внутри интеграции необходима дополнительная работа программиста. Пред которой необходимо получить от клиента список справочников, с конкретными наборами параметров. 
В идеале - получить полную схему данных со всеми связками, чтобы понять структуру данных и набор справочников, который обязателен для интеграции.
