Необходимо создать кабинет для всех клиентов, независимо облачное или коробочное решение. 

Кабинет Smartis создается через сайт https://smartcallback.ru/.

1) Переводим во вкладку "Подключить"

2) Запрашиваем у клиента email, телефон, домен главного сайта или берем из брифа, если на момент создания кабинета он есть. 

Вводим и подключаем.  

3) В кабинете Smartis по названию сайта находим добавленного клиента. 

4) Для быстрого и удобного поиска можно переименовать клиента через шаблонную задачу в консультации. https://app.asana.com/0/1204486395777830/1204827376701614/f
