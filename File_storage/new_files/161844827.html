Бизнес-идея проекта

Чтобы обеспечить достижение финансовых целей по направлению необходимо поменять модель работы с клиентами аналитики. А именно сейчас работа строится от запросов клиентов (не по всей базе, но большей части) при этом формате работы мы зачастую не понимаем какую цель перед перед сервисом ставил клиент на старте, на сколько мы сейчас к ней продвинулись и т.п.. Следовательно не управляем процессом использования продукта. Всё это в конечно счёте влияет на LTV клиента, удовлетворённость сервисом и показатель Churn rate (отток клиентов). Важно понимать, что с учётом быстрых темпов роста клиентской базы в 2023 году будет накоплена критическая масса клиентов, которая может повлечь за собой отток, если мы не поменяем подход в работе с клиентами.

В связи с этим было принято решение о запуске проекта по внедрению модели работы с клиентами, где работа строится на понимание целей, которые ставит перед сервисом клиентом и достижении данных целей.

Бизнес ценность
  1. Churn Rate (отток) - сохранить выручку по ядру клиентской базы. Ядро клиентской базы считаем по абон. счетам 31.12.2022 = 5 762 621 руб.
  2. Увеличение числа кейсов с клиентами отдела аналитики - публичное подтверждение клиентом получаемой ценности от продукта.
Цели проекта

Указанные цели проекта являются промежуточными и ориентированы на результат Q1:

сделана модель работы с клиентом (v1), которая учитывает ключевую цель/цели клиента и работа с клиентом выполняется от цели;
- модель внедрена в работу сотрудников;
- зафиксированы необходимые артефакты (регламенты, инструкции, операции, процессы);
- по итогам первого квартала есть план действий по развитию модели (переход в формат работы QBR и т.д.).

Что получили в результате реализации проекта

Первый этап (Q1):

    1. модель работы с клиентом (версия 1);

    2. выполнена сегментация клиентов (клиенты по которым зафиксирована цель и клиенты по которым мы не можем зафиксировать цель);

    3. по клиентам, которые не имеют зафиксированной цели есть план действий

    4. сотрудники знают/понимают смысл работы с данной моделью и используют в работе (модель отражена в процессах направления);

    5. есть необходимые артефакты (регламенты, инструкции, операции, процессы);

    6. по итогам первого квартала есть план действий по развитию модели (переход в формат работы QBR и т.д.).

Кроме модели работы с клиентом, дополнительным продуктом двух проектов (Разработка и внедрение модели работы с клиентом от целей и оптимизация операционных процессов) по итогам первого квартала должен быть CJM клиента Smartis Аналитика.

CJM клиента Smartis Аналитика - интересен тем, что с точки зрения изучения пути клиента и опыта взаимодействия с нашей компанией мы возможно сейчас учитываем не все проблемные точки. И в дальнейшем эта информация пригодится для инициации других проектов улучшения.

Второй этап (Q2):

Цель второго квартала - внедрение модели работы с клиентом.

Результатом проекта в Q2 будет:

  1. охват клиентской базы - PM работают с 80% клиентской базы по согласованной модели работы;

  2. подведены итоги по клиентам, по которым был старт работы по новой модели в первом квартале. Итоги подведены в формате - цель, результат, профит для клиента, для Смартис;

  3. запушен процесс рекомендаций для клиентов;
