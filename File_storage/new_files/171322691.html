Сотрудники Smartis могут переключаться между ЛК клиентов с помощью интерфейса Клиенты.
В раздел можно попасть по ссылке https://my.smartis.bi/xrm/infopin (облако) или из бокового меню Smartis  - на вкладку "Клиенты"

Для переключения между ЛК необходимо ввести в окно "Клиент"

  • название клиента (как он называется в Smartis)
  • либо id клиента (указывается в скобках рядом с названием после знака #) - id клиентов также хранятся в базе данных

После выбора нужного клиента из списка, необходимо выбрать, в какую оболочку нужно войти:

  • my.smartis.bi - переключит на prod, то есть оболочку, которую используют клиенты Smartis
  • dev.smartis.bi - переключит на dev - оболочку, которая чаще всего обновляется. На этой оболочке чаще всего работают тестировщик, разработчики. 
  • stage.smartis.bi - переключит на stage - оболочку, в которую происходят отгрузки обновлений из dev. На этой оболочке чаще всего работают тестировщик, также желательно чтобы РМ работали также в жтой оболочке. 

Аналогично можно переключаться между оболочками и у клиентов с коробкой, если у них настроено несколько оболочек внутри одной коробки.
