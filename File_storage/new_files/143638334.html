С помощью интеграции "Внешний код" мы можем делать какие-либо доработки, которые нужны только для одного клиента/домена.



Окошечки, куда вставляется код - можно растягивать, чтобы удобнее было работать с кодом.


Как видите, есть множество условий, при которых может выполняться код, а именно:


  1. Код при успешном заказе обратного звонка

callback - код, который будет отработан в случае успешного заказа обратного звонка


       2. Код при завершении подгрузки виджета

По готовности виджета будет исполнен указанный здесь код


       3. Код при открытии виджета
Код выполнится при открытии виджета


       4. Код при закрытии виджета
Код выполнится при закрытии виджета


       5. Код при успешной подписке по Email

Отработает если будет успешно создана подписка через виджет


        6. Код при создании обращения через Smartis.Forms
Код при создании обращения через Smartis.Forms


         7. Код при успешной отправке email-сообщения
Код выполнится, если будет успешная отправка email-сообщения
