Необходимо установить код Smartis на все сайты и на каждую страницу.

Код единый для всех сайтов.


Рекомендации по установке:

  • Код должен быть установлен на все страницы сайта;
  • Рекомендуется ставить в блоке <head> перед закрывающим тегом </head> (можно сразу после кода Google Analytics/Яндекс Метрики);
  • Код асинхронный, так что он не будет блокировать или задерживать загрузку страниц для конечных пользователей;
  • Код единый для всех ваших сайтов и поддоменов, для всех новых сайтов можно будет ставить его;
  • При появлении новых сайтов необходимо уведомлять нас, для регистрации трафика с него в системе Smartis;


Код находится во вкладке "Администрирование" → "Проекты и сайты" → "Код Smartis"
