Как пользоваться этим шаблоном.

Данный шаблон используется как часть процесса развертывания важного продуктового релиза.

Цель процесса: быстрое и полное распространение информации о новом продуктовом релизе.

Результат выполнения процесса:
I. Создан набор материалов для поддержки релиза (УТП, описание, презентация и т.п.)

II. Материалы доведены до сотрудников, проведено обучение по релизу.

III. Разработан и запущен маркетинговый план продвижения релиза.

Описание процесса:

  1. PM (Product Owner) уведомляет PMM (Product Marketing Manager) о предстоящем выпуске важного продуктового релиза.
  2. PMM проводит интервью PM и заполняет документ с описанием продуктового релиза (по этому шаблону).
  3. PMM организует производство маркетинговых материалов.
  4. PMM организует обучение сотрудников.
  5. PMM запускает реализацию маркетингового плана продвижения релиза.
  6. PMM проводит сбор информации о результатах продвижения релиза.
  7. PMM проводит встречу с PM для обсуждения результатов продвижения релиза.

(Wants, Needs, Pains, Fears)

Директор по продажам:

  • Теряю часть сделок. Звонки теряются, часть каналов обращений не обрабатывается или требует сложного процесса контроля.
    • Не понимаю процент пропущенных
    • Долго обрабатываю заявки, когда они уже остыли
    • Не вижу, как общаются с клиентом по мобильным.
  • Неправильно прогнозирую результаты продаж.
    • Не можем отделить нового клиента от повторного.
    • Не могу дать обратную связь маркетингу. Получаю лиды не правильного качества.


Коммерческий директор/Директор по маркетингу, Ответственный за поток входящих лидов

  • Не выполняю план по привлечению лидов
    • Не вижу, откуда приходит более 50% сделок, типо самоходы. В аналитике нет источника сделок, не могу повлиять на рекламу.
    • Неправильно рассчитываю стоимость лида/качественного лида/сделки.
      • Ошибаюсь с повторными лидами.
    • Принимаю решение о изменении в рекламе после завершения кампании и не вижу данных в реальном времени.
    • Не могу экспериментировать с площадками и рекламой.
  • Не могу привлекать обращения на конкретные предложения (размер квартир, локация, примерный бюджет, требования к продукту)
