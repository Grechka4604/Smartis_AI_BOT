Достоверность CRM Endpoint - в случае когда данных "у нас" больше чем "всего" считать достоверность 100%

При передаче данных нам не удаляются сущности, которые в самой CRМ клиента удалены. В связи с этим при подсчете достоверности по CRM Endpoint возникает ситуация, что "у нас" данных больше чем "всего". Из-за этого достоверность была "красной" и мешала выявить ситуации нехватки данных. Поэтому мы сделали доработку для этого клиента, чтобы в случаях когда "у нас" данных больше чем "всего", достоверность всегда была 100%.
