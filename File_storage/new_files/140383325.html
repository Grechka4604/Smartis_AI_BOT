3. Аукцион текстов 

Пример работы: https://id-kudrovo.ru/ 

  • Мы загружаем в систему тексты, при необходимости группируем их по разным категориям, к примеру: Преимущества, Ипотека, Акции, Отделка и т.п. После чего система запускает в тестовом режиме отображение данных текстов и собирает статистику по итогу их показов.
  • Тексты показываются последовательно, в равномерном распределении, т.е. число показов к подведению итогов по всем текстам будет одинаково, как только по каждому тексту набирается 1000 показов система снимает статистику по конверсионности и отдает предпочтение наиболее конверсионному варианту. Таким образом мы полагаемся не на человеческий фактор, какой посыл будет лучше, а используем действительно самый конверсионный текст.

При переходе на сайт застройщика появляется виджет с текстом под его запрос: «Панорамные виды на Павелецкую набережную! Закажите обратный звонок и мы подберем для вас лучший вариант».

11. Работа с Потенциальным клиентом 

  • Человек вводит в виджет номер телефона, но по какой-либо причине не нажимает кнопку «Позвонить». Мы следим, если в течение получаса (это период можно изменять) пользователь так и не оставляет заявку, то мы заносим его номер в лк с пометкой "Потенциальный клиент» 
  • Есть несколько вариантов, как можно этот функционал использовать:
    1. Не прозванивать такие заявки, а только использовать для рассылок, настроек таргета и т.п.
    2. Отправлять уведомления при создании такой заявки с пометкой "потенциальный клиент", чтобы операторы по особому скрипту их обрабатывали
    3. Выделить человека, который через наш ЛК будет запускать на прозвон такие заявки, при этом ему будет доступна карточка со всей историей этого человека и он сможет подготовиться к звонку.

2. Блок «Охват всех коммуникаций»

Основная задача/ценность : Нет данных о первичности таких лидов. Поэтому каждое обращение автоматически фиксируется как первичное. Отсутствие единого пространства для хранения ВСЕХ обращений со всех каналов коммуникации не позволяет адекватно оценить маркетинговые инвестиции

Полный охват всех лидов со всех сервисов  : прозрачность их оценки, автоматическая обработка с минимизацией потери лидов 

Состав решения: 

  1. Автоматизация форм на сайте клиента 
    • Методичка по настройке форм на сайте через Smartcallback
    • Инструкция по автоматической привязке форм (для клиента)
  • Мгновенно реагирует на интерес посетителя, соединяя подготовленного оператора с клиентом
  • Работает с любым встроенным полем ввода
  • Статистика по всем обращениям в одном месте. Все источники по каждому обращению. Просто контролировать работу сотрудников и следить за результатом.
  • Автоматическая повторная обработка заявок, где не состоялось соединение 

2. Автоматизация лидов из соцсетей 

  • Автоматизация лидов из соц.сетей - Интеграция с F.leads
  • Автоматизация лидов из соц. сетей - Интеграция с Vk
  • Подключение интеграции MyTarget
  • Больше не нужно вручную выгружать и прозванивать лиды из каждого рекламного кабинета.
  • Перед началом разговора робот проговаривает оператору все данные о пользователе из соцсети.
  •  Кроме непосредственной автоматизации в виде автозвонка или автоматической подписки на рассылки модуль можно использовать как универсальный интегратор в другие сервисы - CRM, Google Analytics, Яндекс Метрику, Сквозную аналитику и т.д
  • Возможность автоматической верификации посетителя перед перевод звонка оператору

Чаты / Мессенджеры . 

Основная задача/ценность : Дополнительный, актуальный канал коммуникации с клиентом с доступом к переписке, рекламным источником, а также полноценным отчетом для руководителя. 

Единое окно коммуникации по всем мессенджерам и соц сетям для разгрузки КЦ и повышения качества и скорости ответа. 

Состав решения: 

  • Омниканальный виджет для охвата всех каналов коммуникации с клиентом - Чат, whatsapp, Telegram, Viber, F.leads Messenger, Vk Messenger 
  • Информация по источникам обращений через мессенджеры
  • Все чаты и обращения через мессенджеры обрабатываются в едином окне интерфейса
  • Контроль операторов —сохраняется вся история переписки
  • Функционал линейного и обучаемого чат-бота для чатов и мессенджеров для разгрузки операторов
