  1. Помогает структурировать информацию по сайтам/объектам/проектам на первом этапе.
  2. Сформировать структуру отделов и пользователей.
  3. Зафиксировать фильтрацию ключевых показателей воронки.
  4. Зафиксировать кол-во агентств, какие объекты и рекламные кабинеты они ведут. Сформировать привязку 


Отправляется клиенту после трёхсторонней встречи.

Этот файл доступен вам только на чтение, чтобы заполнить данные по вашему клиенту, необходимо:
1. Создать копию этого файла. Вкладка Файл - Создать Копию.
2. Переименовать копию
БРЕНД_Бриф внедрения проекта
3. Заполнить бриф в течение 2х дней с момента получения актуальными, по возможности, максимально полными данными. При возникновении вопросов - напишите вашему менеджеру внедрения.
4. Настроить права доступа на редактирование(!) и прислать в чат по вашему проекту
5. Желтым обозначены все поля, необходимые к обязательному заполнению.
