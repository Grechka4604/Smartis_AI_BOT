Клиент может запросить акт сверки взаимных расчетов (далее — акт сверки) по любому из заключенных с нашей компанией договору.

До постановки задачи в Jira менеджер обязан (зона ответственности менеджера):

- уточнить у клиента по какому договору необходим акт сверки и за какой период времени;

- каким способом клиент хочет получить акт сверки (в ЭДО/курьер или по e-mail).

     1. Постановка задачи (зона ответственности менеджера)

  • менеджер формирует задачу в Jira на специалиста по документообороту (Smartis Документооборот (SMA))

Тип задачи: Клиент SMARTIS

Тема: акт сверки для клиента <наименование юр. лица>.

Юридическое наименование клиента (кратко): <наименование юр. лица>

Тип документа: иной запрос

ИНН: ИНН клиента

Описание: сформировать акт сверки по договору <номер договора> за период <указать период>, способ отправки акта сверки клиенту (ЭДО/курьер* или e-mail)

* обязательно указываются контактные данные: ФИО и телефон лица для получения документа от курьера.

        2. Подготовка акта сверки (зона ответственности старшего бухгалтера)

Специалист по документообороту

  • берет задачу в работу и тэгает старшего бухгалтера

  • в срок не позднее 2 рабочих дней с момента постановки задачи старший бухгалтер формирует акт сверки, вкладывает его в задачу, тэгает менеджера и специалиста по документообороту с комментарием «Акт сверки подготовлен»

Лицо, осуществившее отправку клиенту акта сверки, оставляет в задаче соответствующий способу получения комментарий «Отправлено в ЭДО» / «Отправлено курьером» / «Отправлено по e-mail»

       4. Мониторинг задач (зона ответственности специалиста по документообороту)

Специалист по документообороту осуществляет мониторинг задач:

  • отслеживает в ЭДО подписание акта сверки со стороны 

  • по истечении 5 рабочих дней с момента отправки менеджером акта сверки клиенту по e-mail  тэгает менеджера с комментарием "просьба запросить у клиента скан подписанного акта сверки"

      5. Закрытие задачи (зона ответственности специалиста по документообороту)

  • при отправке акта сверки в ЭДО - после подписания клиентом акта сверки
  • при отправке акта сверки курьером/e-mail - после получения от клиента подписанного экз. акта сверки

!!! Подписанные с обеих сторон акты сверки должны выгружаться в электронный архив
