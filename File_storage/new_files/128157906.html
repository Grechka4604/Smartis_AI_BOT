При работе с входящими/исходящими сотрудник CoMagic Premium в ОБЯЗАТЕЛЬНОМ порядке фиксирует все задачи в пространстве Jira. 


При фиксации задач необходимо строго соблюдать следующие требования: 

  1. Тема задачи - должна отражать основную мысль запроса; 


  2. Описание задачи 
    - что должно стать результатом; 
    если задача состоит из несколько этапов (выполнение работ с тех. специалистами на стороне клиента/нашей, доп. согласования) - план выполнения задачи обязателен в описании; 
    * допускается вставлять в задачу текст письма, если запрос имеет описанные пункты выше. 


  3. У задаче обязательно должен быть согласованный с клиентом срок выполнения
    - если задача не выполняется в установленный срок, то необходимо согласовать с клиентом новые сроки и изменить это в задаче, с соответствующем комментарием; 



  4. Если задача выполняется больше чем один день и состоит из нескольких этапов, то по завершению каждого дня в комментариях фиксируется, что было выполнено за день и какие следующие шаги остались. 
    Итоги дня и следующие шаги прописываются от результата. 
    Например, "делали выгрузку клиенту" - результат не понятен, можно заменить выражением работу работали. 
    Или - "сделали выгрузку клиенту, файл передали клиенту" - это уже лучше. 


  5. Все файлы, которые появляются при работе с клиентом - подгружаем во вложении к задаче
