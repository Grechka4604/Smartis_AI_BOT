Smartis.CDP - стек продуктов, которые используются для активации данных клиента (активация = использование знаний о клиентской базе для оптимизации расходов на рекламу и для улучшения качества обслуживания клиентов).

Что такое CDP-платформа

CDP (Customer Data Platform) - платформа по управлению данными клиента. В рамках кабинета Smartis у наших клиентов хранятся большие объемы данных, например, информация по сделкам, звонкам, визитам на сайт и тд. Клиент может эти данные активировать (то есть использовать в своих рекламных активностях) с помощью нашей платформы. Например, собрать данные о клиентах со сделками и создать аудиторию похожих на них - чтобы найти более целевую аудиторию в рекламных источниках. CDP не связана только с сегментацией пользователей, по сути это инструмент, который помогает также получить информацию о своих клиентах. Эту информацию можно использовать для улучшения знания об аудитории, повышению качества трафика подрядчиков, улучшению клиентского опыта на всех этапах воронки.

Какие варианты использования платформы доступны сейчас

1.Smartis.CDP Конструктор сегментов

в рамках этого решения клиент получает доступ к конструктору сегментов и может сам создавать любое количество сегментов по разным признакам персон, и далее использовать полученные сегменты в произвольных механиках в рекламных кабинетах (на данный момент это Яндекс Директ и MyTarget).

2. Антифрод

Это решение предоставляет функционал скоринга обращений на процент вероятности фрода, то есть помогает определить, с какой вероятностью каждое обращение может быть фродовым. Также, имея данные по каждому обращению, мы выстраиваем аналитику по площадкам для определения качества трафика по каждой рекламной площадке.

  1. В рамках этого решения мы НЕ предоставляем сегменты, решение используется для всех обращений в кабинете клиента - то есть, по всем площадкам клиента.
