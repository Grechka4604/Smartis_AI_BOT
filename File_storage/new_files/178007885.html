Расщепления фильтра "Календарь: Дата отображения" на 2

Зачем это нужно:

Логика маркетинга клиента состоит в том, чтобы оценивать всю воронку рабочих листов в разрезе только тех каналов, которые были зафиксированы до даты создания рабочего листа (МА последнее касание должно отображать канал исключительно перед созданием РЛ). Текущими настройками при использовании календарей этого достичь невозможно


Что сделали:

расщепить фильтр "Календарь: Дата отображения" на 2:

- Календарь: Дата отображения

- Календарь: Дата до которой учитываем касания для моделей атрибуции


Создали фильтр " Календарь: Дата до которой учитываем касания для моделей атрибуции"

до доработки: был фильтр Календарь: Дата отображения по полю "поле даты crm сущности"

после доработки: используем фильтр Календарь: Дата отображения по полю "поле даты crm сущности" + фильтр Календарь: Дата до которой учитываем касания для моделей атрибуции "поле даты crm сущности"

Таким образом, данные в отчете будут отображаться на указанную дату, при этом макретинговые касания (обращения, сессии, заявки, МК) для атрибуции будут учитывться только до даты, которая указана в нужном поле через фильтр Календарь: Дата до которой учитываем касания для моделей атрибуции
