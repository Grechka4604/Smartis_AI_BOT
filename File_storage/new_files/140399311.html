1. Файл по клиентам у которы не закрыты авансы - Авансы по ООО "Смартис" на 28.10.2021
  2. ВАЖНО: до 15.12.2021 нам необходимо по списку клиентов иметь чёткую информацию какие суммы мы возвращаем, а какие закрываем УПД. 
  3. Возможно следующие варианты: 
    - клиент согласен на подписание УПД по авансу в полном объёме; 
    - клиент согласен на подписание УПД по части выполненных работ, т.е. по части аванса; 
    - клиент не согласен подписывать УПД; 
    - клиент не согласен подписывать УПД и требует уменьшения на сумму НДС всех платежей. 

  4. Клиент согласен на подписание УПД по авансу в полном объёме. 
    В этом случае ответственному сотруднику необходимо сформировать задачу в Jira (пространство DOU) на формирование УПД. 
    Шаблон формирования задачи: 
    Тема: Формирование УПД для закрытия аванса ID...... Наименование клиента

    "Добрый день. Коллеги, прошу сформировать УПД из 1С для закрытие ранее оплаченного аванса: 

    ИНН клиента - 9705138170

    ID карточки клиента в Инфопин PP и № и дата письменного договора - ID 310749 SMTP №139 от 03.03.2021

    Наименование товара/услуги - Услуги по внедрению и настройке Сервиса Smartis 2.0

    Вид услуги - Smartis.Аналитика - сервис аналитики

    Сумма - 112 500 руб. 

    Какие требуются документы выставить в 1С - УПД от 31.12.2021
    Подписант - Чичикова Е.А. 
    Пометка для ДОУ - загрузить подписанный скан УПД"


    Далее отмечаем в задаче для согласования 
    После выставления УПД - подписанный скан подгружается в задачу, где вы можете его скачать и отправить клиенту. 

  5. Клиент не согласен подписывать УПД.

    Когда клиент не согласен подписывать УПД вы должны руководствовать правилом: 

    клиент согласен, что стоимость последующих абон. платежей не уменьшится на сумму НДС?
    - если да, то оформляем возврат оплаченного аванса в полном объёме и предупреждаем клиента, что платёж необходимо будет оплатить в полном объёмме после новогодних праздников;
    - если нет, возвращаем только сумму НДС из авансового платежа; 

    - если думает, то возвращаем полность сумму аванса. 

    Как вернуть аванс (в полном объёме или частично). 

    - запросить у клиента в письме реквизиты для возврата аванса. 

    Шаблон письма. 

    "Добрый день. Коллеги, для осуществления возврата ранее оплаченного интеграционного платежа в пользу ООО "СМАРТИС" (ИНН 7727428345) в размере <указать сумму с копейками>

    просим вас в ответном письме направить реквизиты для перечесления." 

    - после получения письма с реквизитами сформировать задачу в Jira (пространство DOU) на осуществления возврата. 

    Шаблон формирования задачи: 
    Тема: Возврат ранее оплаченного аванса ID...... Наименование клиента

    "Добрый день. Коллеги, прошу осуществить возврат ранее оплаченного аванса: 

    ИНН клиента - 9705138170

    ID карточки клиента в Инфопин PP и № и дата письменного договора - ID 310749 SMTP №139 от 03.03.2021

    Сумма возврата: ........" 
    В задачу вкладываем письмо клиента с реквизитами для возврата. 

    Далее отмечаем в задаче для согласования 

  6. Клиент согласен на подписание УПД по части выполненных работ, т.е. по части аванса
    В этом случае вам необходимо сформировать две задачи: 
    - согласно п.4 (выше) на формирование УПД; 
    - согласно п.5. на возврат оставшейся суммы; 


  7.  Клиент не согласен подписывать УПД и требует уменьшения на сумму НДС всех платежей. 
    Формируем задачу на возврат ранее оплаченной суммы НДС. Сумму НДС можно взять из файла п.1. 
    Задачу формируем согласно пункту 5.
