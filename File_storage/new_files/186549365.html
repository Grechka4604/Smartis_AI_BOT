Видео


Цель - упростить процесс внедрения благодаря описанной последовательности задач, и сократить сроки внедрения до 2х-3х месяцев. 

Владелец процесса: Project менеджер внедрения.

Инициатор: Project менеджер внедрения.

Инфо поле: Asana.


Алгоритм создания клиента в asana. 

1) Когда появляется новый клиент необходимо создать портфель в асане с названием клиента, через вкладку "Портфели".

*Возможно проект уже существует и необходимо его найти. 

2) Необходимо создать шаблонный проект внедрения. 

Актуальный шаблон внедрения облачного решения находится по ссылке https://app.asana.com/0/1206889317970438/1206896943785756 https://app.asana.com/0/1206906335175460/1206908357445482

Для использования шаблона дублируем проект, далее переименовываем на "ID / Название клиента / Внедрение".

3) Добавляем в портфель клиента новый проект внедрения.

4) Создаем основной проект клиента по шаблону https://app.asana.com/0/1206889317970446/1206897054147969 с наименованием "Бренд/ Основной проект" и добавляем в портфель клиента.

В основном проекте по клиенту и в проекте по внедрению мы фиксируем информацию после внедрения по шаблону Шаблон карточки клиента
