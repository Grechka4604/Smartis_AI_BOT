Бизнес-задачи collab.ru

1. Поиск и Устранение bottlenecks воронки сотрудниками collab.ru.

- Задача как минимум в 2 раза повысить конверсию из заполненной анкеты до начала получения услуги (сейчас 1/3 → эксперименты до 44%)

2. Аналитика по сотрудникам:

- Эффективность стилистов, KPI: выручка, средний чек, NPS (это потом).

- Администраторы, промоутеры - мотивация на привлечение клиентов, за запись на услугу - фикс плата. В Smartis - видеть по фактическим записям расход на вознаграждения

3. Продуктовая аналитика

- по брендам,

- товарным категориям,

- Размеры/фасоны (свойства товаров).


2. Задачи перед Smartis:

2.1 Cобрать показатели необходимые для решения бизнес-задач выше

2.2 Соединить CJM клиентов внутри данных Smartis

2.3 Обучить команду collab.ru аналитике в Smartis (в первую очередь конструктор отчетов, а не дашборды) для решения их бизнес-задач
