Бизнес идея:

(боль) проекта

Емкость текущего сегмента "Недвижимость. РФ" конечна, по прогнозу на старсессии 2023Q3, остаточная емкость составляет ~ 5 млн абонетских платежей и скорость забора этой есмкости ~ 1,5 года.

Smartis исследует гипотезы нового канала продаж или сегмента.

На текущий момент имеем 11 бизнес-гипотезы, проекты развития (https://callgeardmcc.sharepoint.com/:x:/r/sites/Smartis-/_layouts/15/doc2.aspx?sourcedoc=%7B86438ad2-c8a5-48f4-8bf5-380c16cd10a0%7D&action=edit&activeCell=%27%D0%9B%D0%B8%D1%81%D1%821%27!K11&wdinitialsession=6cd949fe-4890-4df4-ac7b-bbd5f888ad28&wdrldsc=5&wdrldc=1&wdrldr=AccessTokenExpiredWarning%2CRefreshingExpiredAccessT)

Самой перспективной для исследования в 2024Q1 выбрано исследование возможности захода на рынок Дубая с текущим РФ решением.

Цели проекта

Что команда должна получить по завершении проекта?

Возможность репликации аналитического решения Smartis на новый сегмент

Возможность обкатать решение для запуска в других странах с агентской моделью продаж.

Задачи проекта

Действие, которое необходимо совершить для достижения вашей цели

На первую часть проекта/2024Q2): РФ застройщик заходящий в Дубай (знакомство на Движении).

Отработать дубайский форум Движение

1. Согласование участия в Форуме Движение

2. Определить операционные цели участия в Форуме

3. Получить список участников и создать бэклог в отдельной воронке под проект в АМО CRM

4. Провести подготовительную коммуникацию из Москвы с потенциальными участниками форума

5. Провести встречи с ЛПР как на полях форума, так и после


На вторую часть проекта/2024Q2: РФ агентство заходящее в Дубай (выбирать среди действ клиентов CallGear)

Продукт проекта

Инструмент для достижения цели проекта

Работающий процесс репликации аналитического решения Smartis на новый рынок
Выгода для компании

  1. Увеличение прибыли: большую прибыль, расширение клиентской базы и увеличение объема продаж.

  2. Занять нишу аналитических решений на международном рынке (в данном случае Дубай)

  3. Диверсификация бизнеса. Возможность репликации решения.
