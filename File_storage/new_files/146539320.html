На данной странице прописаны основные правила оформления проектов. 

Логика работы с проектом разобрана по вкладкам - обзор, список. 

  1. Вкладка обзор должна быть заполнена в "Основном проекте" следующим образом

Контактные лица + контакты 

Содержит следующую информацию

  • ФИО
  • Телефон/ны (при наличии)
  • Почта
  • Должность 
  • Зона ответственности

ID инфопин+xсrm

  • ID 
  • XCRM

Информация о проекте

  • Особенности по проекту, договорённости, а так-же прочая инф., которую важно учитывать в работе
  • Ключевые метрики
  • Цели проекта

Роли в проекте

  • Добавлены сотрудники, которые участвую в проекте и прописаны роли

Основные ресурсы

  • ключевые документы/ссылки, которые необходимо использовать в работе

Статус проекта

  • заполняется еженедельно согласно правилам заполнения статусов описанных в разделе, каждый понедельник синхронизируемся по статусам


2. Список - задачи, которые ведутся по клиенту. 

В вкладке "список" обязательны должны быть выделены следующие разделы

  • в работе 
  • в ожидании 
  • отложено
  • дебиторка (оттчёты, счета, УПД)
  • ежемесячные задачи 
  • задачи на разработку 
  • задачи разработки в работе 
  • документы (договора, спецификации, переоформление)

Настройка столбцов 

  • Исполнитель
  • Срок выполнения
  • Priority
