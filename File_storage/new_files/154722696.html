Документ описывает какие действия необходимо выполнить, чтобы "упаковать" продуктовые материалы для нового продукта и провести его запуск.

Цель - создание продуктовых материалов (для сотрудников) необходимых для вывода продукта в рынок (запуск продукта в рынок)

Участники процесса: PMM (Product Marketing Manager), PM (Product Manager), Руководители направлений 

Инициатор: PM/ BO (Business Owner)/ Руководитель направления

KPI процесса: продукт подлежащий упаковке упакован или по нему есть план по конкретным шагам для достижения упаковки в обозримые сроки

Как контролируем: ???

Результат: 

1. Создан набор продуктовых материалов ("поставить телегу на колеса").

2. Валидирована правильность использования и эффективность материалов ("убедиться, что телега поехала").

Владелец процесса: PD (Product Director) ?

Статус процесса: бета-тестирование.


Описание процесса:

При инициации процесса необходимо:

  1. Проверить, пройден ли уже с данным продуктом трекшен-меп (левая его часть) и удостовериться, что:
    1. Продажа (как минимум одна).
    2. Выполнено успешное внедрение продукта клиенту.

    3. Есть полученная ценность (клиент подтверждающий ценность от продукта)

  2. Определить задачу, для которой запускается процесс:
    1. Запуск продаж новым клиентам.
    2. Запуск допродаж действующим клиентам.
    3. И то, и другое.

Этапы процесса

  1. PM готовит вводную информацию (начальная информация в матрице работ) 
    1. УТП продукта в описании работа[когда хочу чтобы], чья она, какое решение
    2. Кейс (передается отделом маркетинга)
    3. Калькулятор
  2. PMM и PM/PD проводят встречу по передаче вводной информации и формированию плана по упаковке материала.
  3. PMM делает набор артефактов по списку (ниже)
    1. Ключевой продуктовый материал (презентация; сценарий демо реализованный в демо-кабинете; и то и другое).
    2. Сценарий продажи + обучение по сценарию (кто отвечает за обучение, кто проводит обучение) 
    3. Тарификация  (включение тарификации в артефакты системы продаж [калькулятор, КП], обучение и аттестация по работе с тарификацией)
    4. Кейс (переработка в формат, необходимый для использования в продажах).
    5. Сравнение с конкурентами  [опционально - необходимость определеяется на старте задачи]
  4. PMM+PM проводят обучение отдела продаж (+маркетинг + аккаунты поддержки - в зависимости от цели)
  5. PMM валидирует правильность и эффективность использования продуктовых материалов.
