Команды чувствительные к регистру букв! (большие/маленькие)


SCBopen(1) - открытие виджета
SCBopen(1,'call') - открытие Mobile Direct
SCBopen(1,'mail') - открытие виджета с подпиской (email)
SCBopen(1,'taxi') - открытие виджета такси
SCBopen(1,'telegram') - открытие telegram
SCBopen(1,'whatsapp') - открытие whatsapp
SCBopen(1,'viber')- открытие viber
SCBopen(1,'F.leads') - открытие F.leads
SCBopen(1,'vk') - открытие ВКонтакте
SCBopen(1,'liveChat') - открытие окна чата (большое окно чата)
SCBopen(1,'chat') - открытие окна чата (маленькое окно чата)
SCBopen(1,'messengers') - открытие окна с мессенджерами


SmartCallBack.vars - выводит информацию о номере версии, номере правила, id виджета 


dat.di - выводит информацию о  id клиента в СКБ;  id домена в СКБ, id виджета


jSCB('наименование_селектора').length - с помощью этой команды проверяем, можно ли использовать селектор при подвязке форм. 
Должен выдавать цифру "1". Если выдаёт что-либо другое - селектор использовать нельзя


jSCB('наименование_селектора').val() - с помощью этой команды проверяем, тянутся ли данные с этого селектора.
