Что необходимо клиенту (запрос)? 
Дашборд Анализ эффективности менеджеров отдела продаж позволяет: 

  • Оценить эффективность работы менеджеров на каждом этапе воронки; 

  • Оценить конверсию по менеджерам на каждом этапе воронки; 

  • Оценить нагрузку на менеджера по дням/неделям; 

  • Увидеть воронку в разрезе объектов. 

 
Зачем это нужно клиенту, в каком процессе это участвует? 

  • Есть потребность в том, чтобы устанавливать KPI менеджерам; 

  • Чтобы подумать над мотивации сотрудникам; 

  • Оптимизация процессов: влияние на конечную цель-прибыль. 

 
Техническое решение: 

Это готовый дашборд, есть в демо-клиенте. Настраивается на внедрении. Подробнее почитать о нем можно здесь.


Описание готового решения: 

       1. Эффективность менеджеров продаж по воронке 

 

В отчете используются 5 показателей, которые опираются на каждый из этапов воронки. Отчет показывает: 

  1. сколько визитов в офис продаж инициировал каждый из менеджеров 

  1. сколько договоров бронирования подписал каждый из менеджеров 

  1. сколько ДКП было подписано каждым менеджером 

  1. а также конверсию в бронирования и конверсию в продажу 

 

       2. График ТОП-менеджеров по эффективности 

 

Отчет показывает конверсию из визитов в продажи в разрезе каждого менеджера в графике. Быстро, легко, наглядно.


       3. Общая выручка с менеджера за период 

По отчету можно посмотреть сколько денег принес в компанию каждый менеджер в кольцевой диаграмме (с процентами и суммами).


       4. Объемы продаж по объектам 

 

Этот отчет показывает на какую сумму проведены продажи по каждому из ЖК за выбранный период.


       5. Обращения/Визиты в офис и продажи по дням недели 

 

В разрезе дней недели отчет показывает статистику сколько обращений, визитов и ДКП было за определенный период.


       6. Нагрузка на менеджеров в течение дня по часам 

Отчет рассказывает нам какая нагрузка оказывается на офис продаж в течение дня по часам.
