Ниже прописаны базовые отчеты, которые мы настраиваем на внедрении.


1.Отчет Эффективности маркетинга.

 Дашборд сквозной отчет по всем каналам необходим для:

  1. Медиапланирования на сквозных данных;

  2. Отслеживания эффективности каналов/площадок с детализацией до кампаний/объявлений;

  3. Отслеживание конверсий на всех этапах воронки;

  4. Отслеживание стоимости действий клиента на всех этапах воронки;

  5. Автоматизирования подсчета целевых метрик- целевых обращений, визитов, броней, сделок. 



Состоит из одноцифровых виджетов основных показателей воронки: 

  • Целевые обращения;

  • CPL: Целевого обращения;

  • CR%: Целевого во встречу;

  • CRM: Встречи;

  • Стоимость Встречи;

  • CRM: Брони;

  • Стоимость Брони;

  • CRM: Успешные сделки;

  • Стоимость Сделки;

  • %ДРР, ROMI


Сквозной отчет по всем каналам, можно строить в разной модели атрибуции.

Состоит из показателей:

  • Бюджет факт с НДС и АК;

  • Посетители;

  • Валовые обращения;

  • CR%: Валовых обращений в Целевые обращения;

  • CRM: Целевые обращения;

  • CPL: Целевого обращения;

  • CR%: Из целевого в визит;

  • CRM: Встречи;

  • CR%: Из Встречи в Брони;

  • CRM: Брони;

  • CR%: Из Броней в Сделки;

  • CRM: Успешные сделки;

  • CRM: Стоимость сделки;

  • CRM: Сумма сделок;

  • %ДРР;

  • ROMI. 






2. Аналитика обращений

Дашборд аналитика по обращениям необходим для:

  1. Анализа выполнения плана по целевым обращениям;

  2. Отслеживания стоимости трафика, всех обращений и целевых обращений;

  3. Отслеживания динамики данных по дням;

  4. Отслеживание конверсий по этапам посещения/обращения/целевые;



Одноцифровые виджеты состоят из:

  • Посещений;

  • Стоимости платного трафика;

  • Валовых обращений;

  • CPL: Валовых обращений;

  • CRM: Целевые обращения;

  • CPL: Целевые обращения. 





Динамика данных состоит из:

  • Посещений;

  • Валовых обращений;

  • Целевых обращения. 




Обращения по каналам состоят из:

  • Посетителей;

  • CR%: Посещений в Валовые обращения;

  • Валовые обращения;

  • CPL: Валовых обращений;

  • CR: Валовых обращений в целевые обращения;

  • Кол-во целевых (план)

  • CRM: Целевые обращения;

  • CPL: Целевого обращения;

  • %Выполнение плана по целевым обращениям.


Так же виджет с такими же показателями, только в разрезе объектов. 



По необходимости можно добавить график динамика по сервисам обращений (звонки. заявки, чаты), Валовые обращения по регионам и график по стоимости обращения по каналам.








3. Анализ эффективности контекстной рекламы

Дашборд Анализ эффективности контекстной рекламы необходим для:

  1. Более детального анализа контекстной рекламы от показов до суммы сделок;

  2. Анализа эффективности данных в разрезе кампаний/объявлений/ключевых слов. 

  3. Отслеживания динамики данных по дням;

  4. Анализа эффективности данных в разрезе категорий кампаний (бренд/не бренд, поиск/РСЯ, Гео и т.д.)



Одноцифровые виджеты состоят из:

  • Расходов ЯД;

  • Показов ЯД;

  • Кликов ЯД.

Динамика данных состоит из:

  • Расходов ЯД;

  • Валовых обращений;

  • Целевых обращений.



Воронка в разрезе поиска/РСЯ, Бренд/Не бренд (и другие категории кампаний по желанию) состоит из показателей воронки:

  • Расхода ЯД;

  • Показов ЯД;

  • Кликов ЯД;

  • Валовые обращения;

  • CPL: Валовых обращений;

  • CRM: Целевые обращения;

  • CPL: Целевого обращения;

  • CRM: Встречи;

  • CRM: Брони;

  • CRM: Успешные сделки;

  • CRM: Сумма сделок;

  • %ДРР.


Данные отчеты так же строим в разрезе кампаний и объявлений. 

В каждом виджете можно с помощью проваливания детализировать данные до кампаний/объявлений/ключевых слов.

5. Эффективность менеджеров отдела продаж

Дашборд Анализ эффективности менеджеров отдела продаж позволяет:

  1. Эффективность работы менеджеров на каждом этапе воронки;

  2. Конверсию по менеджерам на каждом этапе воронки;

  3. Нагрузка на менеджера по дням/неделям.

  4. Воронка в разрезе объектов.


Виджет эффективность  менеджеров по воронке состоят из показателям:

  • Целевых обращений;

  • CRM: Визиты в офис;

  • CR%: Визиты -> Брони;

  • CRM: Брони

  • CR%: Брони -> Сделки;

  • CRM: Сделки;

  • CRM: Сумма сделок. 



Виджеты нагрузка менеджеров по часам и обращения по дням неделям.

Состоят из показателей:

  • Все обращения;

  • Целевые обращения;

  • CRM: Визиты;

  • CRM: Сделки.


Виджет Воронка по объектам.

7. Бюджет План/Факт

Дашборд Бюджет план факт помогает:

  1. Контролировать План и факт  по расходам в разрезе площадок;

  2. Анализировать процент выполнения по маркетинговым расходам;

  3. Контролировать план - факт выполнения по стоимости целевого обращения, визитов, броней, сделок. 


Виджеты состоят из показателей:

  • Бюджет план с НДС и АК;

  • Бюджет факт с НДС и АК;

  • % Выполнения плана по маркетинговым расходам. 

  • Кол-во целевых обращений (План);

  • CRM: Целевые обращения;

  • % выполнение плана по целевым обращениям.


… и план/факт по другим показателям, которые необходимо анализировать по БП клиента.
