2+РаботаетОП
  1. Проведено интервью с возможным числом клиентов, по итогу месяца.
    Подведены итоги, сформулированы гипотезы улучшений.
    Проведено обсуждение итогов и гипотез улучшений с командой руководителей.
  2. По блоку интервью предыдущего месяца запущен тест гипотез.
  3. По предыдущим тестам гипотез подведены итоги.

    Фиксация: записи встреч, ключевые результаты и гипотезы зафиксированы в таблице. Рук-ль ОП принял гипотезы и дал полномочия на работу по ним.

21РаботаетPD
  1. Проведена сверка по итогу обновлений за месяц с PD
  2. Обновления зафиксированы в разделах где находятся материалы (конф)
  3. Обновления из перечня доставлены к руководителям
  4. У руководителей нет вопросов к обновлениям или зафиксированы и есть план по устранению

    Фиксация: подтверждение рук-лей, что все ок.

33РаботаетPD
  1. Выполнен заявленный на завершившийся месяц план по упаковке (перечень материалов и задач по упаковке).
  2. Сформирован и согласован с командой продаж и маркетинга план по упаковке на текущий месяц.
