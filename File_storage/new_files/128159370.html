1. Тест нового личного кабинета и Отключение номеров от неактивных посетителей

Цель обсуждения
  1. Как говорили на встрече, добавляем клиента в тестовую группу по тестированию нового личного кабинета. По срокам сообщу чуть позже, когда обговорю с разработчиками.
  2. Выслала информацию по опции "Открепление номеров от неактивных посетителей сайта".
    С помощью этой опции клиент сможет сэкономить на постоянной покупке номеров по РК с динамическим коллтрекингом в случаях, когда нехватка номеров вызвана не увеличением трафика, а пиками.

    Как она будет работать: на сайт приходит посетитель, мы через несколько секунд проверим, ушел он с сайта или нет. Если ушел, то, когда динамических номеров будет не хватать - мы заберем его номер раньше времени резерва.
    Время проверки (т.е. кол-во этих нескольких секунд) считается индивидуально для каждой РК. Опция подключается индивидуально, исходя из статистики каждой рекламной кампании. То есть, ее может не получится подключить, если по РК очень мало звонков на динамику.
    Опцию не получится подключить, если:
  • Только 1 номер в пуле ДТ;
  • По РК за последние 7 дней каждый день был охват = 100%;
  • По РК не было звонков на динамические номера за последние 7 дней
