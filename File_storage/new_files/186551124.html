В Smartis существует специальный фильтр "Smartis.CRM: Порядковый номер среди всех - #8452". Для его работы требуется заполнение кастомного поля "Номер элемента среди всех этого типа" для всех CRM элементов. Это делается в синхроне #126 "SmartisCRM: Анализ MainParams". Для БМ Групп существуют определенные ограничения (заполнение кастомного поля осуществляется только для Звонков и Встреч).

См. код приложения App\Synchronizers\Stat\SmartisCRM\CrmElementsMainParams::run и App\Classes\ModelAttributions\Synchronizers\CrmElements\CrmElementsMainParams::get
