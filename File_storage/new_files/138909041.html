1. Выгружаем потенциальных клиентов по запрошенному периоду (как правило это 1 месяц)
2. Находим в выгрузке колонку “телефон” и копируем все номера в отдельный файл exel
3. В колонке с номерами нужно проставить запятые, делаем это через формулу =B1&","  

 (нажимаем enter), далее растягиваем до конца номеров 
4. Копируем номера из второй колонки и вставляем их, как значение через специальную вставку 

Получаем номера без формулы (для заливки в Smartis)

Открываем кабинет клиента в my.smartis.bi и чтобы отфильтровать лишние номера, которые уже обработаны, строим отчёт и используем указанные на скриншотах фильтры. (скриншот ниже)

1. Данный отчёт покажет нам кол-во номеров потенциальных клиентов, которое уже обработано и не требует доп. обработки клиентом.

Получив цифру таких номеров нажимаем на неё и проваливаемся в базу обращений, откуда делаем выгрузку этих номеров - нажимаем на ЭКСПОРТ
1.1 из полученной выгрузки вырезаем номера в отдельный файл exel и подставляем им запятую, чтобы не делать это вручную при заливке в смартис.



2. Строим отчёт исключая обработанные номера и получаем готовый список потенциальных клиентов, который и нужен нашему клиенту.
Для этого из второй выгрузки вставляем номера через запятую, выбираем указанные на скриншоте фильтры и получаем готовый отчёт. 
Финальную выгрузку отправляем клиенту. (скриншот на сл. странице)
