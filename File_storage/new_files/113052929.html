Smartis - состоит из нескольких подсистем:

  1. Личный кабинет облачного решения - https://my.smartis.bi
    У    прощенный встроенный Bi инструмент решающий задачи маркетологов, топ-менеджеров коммерческого блока, аккаунт-менеджеров рекламных агентств (важно, что не аналитиков - Smartis не позиционируется как всемогущая bi-платформа как Tableau или Power bi). 

  2. (Extract, Transform, Load)
    Можно считать ядром продукта. Собирает данные из всех подключенных клиентами внешних сервисов, обрабатывает их, анализирует, сохраняет в сыром и полуагрегированном состоянии для быстрых отчетов в DWH.
    Здесь происходит магия анализа многоканальных последовательностей, поиск персон, контролируется достоверность данных, ведется поиск аномалий в данных клиентов и т.д. 
    Состоит из нескольких серверов, которые можно добавлять по мере необходимости, тем самым увеличивая объемы и скорость обработки данных.

  3. Data Ware House (DWH) - хранилище данных
    Несколько хранилищ данных разного типа (MySQL, Clickhouse, Redis, files) объединенных единой ETL системой. Хранит все данные всех сервисов в сыром виде и с добавлением дополнительных свойств (рекламный канал, рекламная площадка, рекламное агентство, персона, даты первых касаний итд) необходимых для большей пользы и скорости аналитики
  4. js-счетчик для сайтов
    Одна строчка кода на сайте клиента позволяет сохранять сессии и поведение пользователей в сыром (в отличие от Google Analytics и Яндекс Метрики) виде. Дополнительно выполняет роль кросс-доменной связки пользователей в алгоритме Анализа персон
  5. Репозиторий разработчиков http://mydev.smart-is.ru/
    Зеркало всего продукта, на котором выкатываются релизы перед поставкой их клиентам. Тут можно тестировать новые функции. 
  6. Коробки клиентов
    Локальные копии продукта в серверном контуре клиентов которые купили on-premise версии. Они имеют свой замкнутый комплекс из описанных здесь подсистем за исключением 5 и 7, и временно 4.
  7. Внутренняя система управления клиентами - https://xrm.smartcallback.ru
    Минимально необходимый функционал для создания, поиска клиентов, распределение между аккаунт-менеджерами, входа в личные кабинеты клиентов в один клик
