Как поставить задачу в Jira для подготовки договорного документа по клиенту (зона ответственности инициатора)


Инициатор

  • формирует задачу в Jira на специалиста по документообороту (Smartis Документооборот (SMA))

Тип задачи: Клиент SMARTIS

Тема: * <наименование юр. лица>

Юридическое наименование клиента (кратко): <наименование юр. лица>

Тип документа: договорной документ

Описание: *



* заполняются в соответствии с регламентами (заключение договора, расторжение договора, переоформление договора, подготовка Спецификаций к действующему договору с клиентом)
