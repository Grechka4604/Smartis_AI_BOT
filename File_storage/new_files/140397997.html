У нас есть возможность провести интеграцию с чатами auto.ru

Они выглядят следующим образом:





Посетитель обращается в чат на сайте auto.ru → это обращение попадает в наш раздел чаты/соц. сети, где в свою очередь с ним работает оператор КЦ застройщика


При этом, сначала эта информация попадает в кабинет SmartCallBack, а затем с помощью синхрона между двумя кабинетами, попадает в кабинет Smartis.


Первоначально, необходимо зарегистрировать клиента в Blinger (если его еще нет), пройти первые 8 пунктов из этой документации - 

Если подключение №1 уже занято в кабинете SmartCallBack - мы подключаем всё на второе подключение! Вебхук у подключений разный, перепутать нельзя!


Далее, нужно создать кастомное приложение на стороне Blinger:

заходим в сам Blinger → раздел "Приложения" → нажимаем "Добавить" → находим Custom App, нажимаем добавить



Заполняем имя;

заполняем Custom App ID - придумываем сами; 

webhook url - берем из настроек СКБ (пункт №6 в документации )


нажимаем добавить




Создаётся кастомное приложение.

После этого необходимо зайти в редактирование кастомного приложения, для этого находим созданное кастомное приложение и нажимаем "изменить"



Берем отсюда следующие параметры:

-Токен

-Custom Channel Id

-Webhook url to Blinger


и вставляем их в настройки Чаты и соцсети, в то подключение, на которое создавали кастомное приложение, сохраняем. 




Далее, в настройках домена, на который мы подключаем интеграцию - нужно найти интеграцию с auto.ru. 

Нажимаем на шестеренку, вносим:

-ChatID, который прописали на стороне авто.ру. Можно придумать любое сочетание букв, цифр и т.д. Главное, чтобы ChatID совпадал в настройках интеграции СКБ и на сайте auto.ru. 

-выбираем тип подключения, где настроили кастомное приложение и сохраняем.



Не забываем включить интеграцию! 


Интеграция настроена! 


Если по auto.ru от вас просят вэбхук, то он выглядит вот таким образом:
https://my.smartis.bi/scb/api/{apiToken}/messengers/{domainId}/userMessage/auto-ru

{api_token} - это АПИ токен клиента СКБ
{domain_id} - идентификатор домена куда будут складываться заявки.

Создаёте доступ в https://xrm.smartcallback.ru/api-client/ и берёте оттуда данные (api_token). 

На стороне авто.ру нужно внести URL для вебхука и Chat-id
