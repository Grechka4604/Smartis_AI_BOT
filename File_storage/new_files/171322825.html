Бизнес задачи:

Чтобы оценить какой сайт более конверсионный и вести на него больше трафика
Чтобы оценить какая из форм Марквиза/Флексби более эффективная (с 2022 начали формы квизов делать в виде посадочных)

При фильтрации по конкретному объявлению - выбрать для него сайт-посадочную с наибольшей конверсией в целевое действие


Как работает:

Для обращений - на основе свойства "Страница входа" (requests.first_page), очищая от www, http://, внутренних страниц сайта и GET-параметров (то идет после ?)
Для сессий - на основе свойства "Страница входа" - domain(first_page_path) - функция кликхаус, которая так же очищает от от www, http://, внутренних страниц сайта и GET-параметров (то идет после ?)
Для просмотров медийной рекламы - на основе свойства "Страница куда ведет баннер" (view_banner_link)
Для CRM - через анализ "на лету", раскладывая на свойствах тех касаний описанных в п1-3
Для показателей импорта - берет наименование сайта на который импортированы данные


Бывает удобно использовать эту группировку в комбинации с фильтром "Страница входа (посадочная страница)", чтобы получить необходимый отчет по конкретным посадкам.
