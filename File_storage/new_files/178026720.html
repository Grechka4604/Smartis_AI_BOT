Потребность в подготовке Спецификаций к действующему договору с клиентом возникает в случае, когда клиент подключает дополнительные услуги (услуги CDP/Рost-view).

До постановки задачи в Jira менеджер обязан (зона ответственности менеджера):

- согласовать с клиентом дату начала оказания дополнительной услуги (дату спецификации).


1. Постановка задачи (зона ответственности менеджера)

  • менеджер формирует задачу в Jira на специалиста по документообороту (Smartis Документооборот (SMA))

Тип задачи: Клиент SMARTIS

Тема: подготовка спецификации <наименование юр. лица>

Юридическое наименование клиента (кратко): <наименование юр. Лица>

Тип документа: договорной документ

Описание: дата начала оказания дополнительной услуги (дата спецификации), стоимость дополнительной услуги, способ подписания документа (ЭДО/курьер*)

* обязательно указываются контактные данные: ФИО и телефон лица для получения документа от курьера.

2. Подготовка документа (зона ответственности специалиста по документообороту)

Специалист по документообороту

  • берет задачу в работу (переводит задачу в статус «Подготовка документа»)

  • формирует документ по шаблону, вкладывает его в задачу и тэгает менеджера с комментарием «Отправить клиенту документ на согласование» и переводит задачу в статус «Согласование документа»

  • менеджер отправляет сформированный документ на согласование клиенту

3. Согласование документа (зона ответственности менеджера)

  • менеджер организует согласование документа с клиентом

  • после согласования документа с клиентом менеджер тэгает специалиста по документообороту

4. Подписание документа (зона ответственности специалиста по документообороту)

  • после согласования документа с клиентом специалист по документообороту переводит задачу в статус «Подписание» и организует подписание документа (аналогично процессу → Заключение договора с клиентом)

5. Закрытие задачи (зона ответственности специалиста по документообороту)

  • после получения подписанного документа в системе ЭДО/оригинала документа в бумажном виде (в случае, если подписание документа осуществлялось сторонами не посредством ЭДО) специалист по документообороту формирует отчет в задаче, тэгает менеджера с комментарием «Задача выполнена», размещает скан документа в электронном архиве, закрывает задачу
